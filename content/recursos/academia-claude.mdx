## Qué es Anthropic Learn

La academia oficial de Anthropic — la empresa que hace Claude. Cursos **gratuitos**, algunos con **certificado oficial**. URL: `anthropic.com/learn` (cursos hosteados en `anthropic.skilljar.com`).

Diseñada para todos los niveles. **No necesitás saber programar** para empezar los básicos.

| Característica | Detalle |
|----------------|---------|
| Costo | Gratis |
| Certificados | Sí (en cursos seleccionados) |
| Tarjeta de crédito | No requerida |
| Niveles | Principiante / Intermedio / Avanzado |
| Idiomas | Inglés principalmente (algunos sub en español) |

---

## Los 2 cursos clave por dónde empezar

### Claude 101

> Curso fundamental. Aprende qué es Claude, cómo funciona, cómo usarlo en tu día a día.

URL: `anthropic.skilljar.com/claude-101`

Duración: ~2-3h. **Vale incluso si ya usás Claude**. Hay matices que probablemente no estás aprovechando (extended thinking, tool use, prompt structure).

### Claude Code in Action

> Curso práctico de Claude Code. Usarlo como copiloto de desarrollo para construir, debuggear, automatizar.

URL: `anthropic.skilljar.com/claude-code-in-action`

Duración: ~4-6h. **Vale si usás Claude Code intensivamente**. Cubre flows que la documentación oficial no detalla tanto.

## Rutas por nivel

### Principiante — "nunca usé IA seriamente"

```
1. Claude 101
2. AI Fluency
3. Claude for Personal
```

Tiempo total: ~10h spread en 2-3 semanas. Al terminar:
- Entendés cómo funciona un LLM por dentro
- Sabés escribir prompts decentes
- Tenés casos de uso para tu vida diaria

### Intermedio — "uso Claude pero quiero más"

```
1. Claude for Work
2. Skills
3. Claude Code in Action
```

Tiempo: ~15h. Al terminar:
- Sabés implementar Claude en workflow profesional
- Entendés el sistema de skills (no solo plugins genéricos)
- Tu uso de Claude Code se vuelve 2-3x más eficiente

### Avanzado — "quiero construir cosas serias"

```
1. Build with Claude
2. APIs & SDKs
3. MCP
4. Agents
```

Tiempo: ~25h + práctica. Al terminar:
- Podés construir apps con la API
- Entendés cuándo usar prompt vs tool vs agent
- Sabés diseñar agentes multi-step con MCPs

## Qué hay de bueno y qué no

### Pros

- **Oficial de Anthropic** — info correcta y actualizada
- **Gratis sin trucos** — no upsell oculto
- **Certificados serios** — valen en CV/LinkedIn
- **Práctica real** — ejercicios hands-on, no solo teoría
- **Updated** — actualizan según releases (Sonnet, Opus, features nuevas)

### Cons honestos

- Mayoría en **inglés** — subtítulos sí, dub no
- Algunos cursos asumen **stack específico** (Python para APIs)
- **No reemplaza experiencia** — completar todo no te hace senior
- Comunidad menos activa que cursos pagos (Udemy/Coursera tienen Q&A más vivo)

## Ruta que recomiendo (en orden)

Si tuvieras que hacer **una sola pasada** balanceada:

```
Semana 1-2: Claude 101 (gratis, 2-3h)
Semana 3: Prompt Engineering course (gratis, 4h)
Semana 4-5: Claude Code in Action (gratis, 4-6h)
Semana 6: MCP course (gratis, 3h)
Semana 7-8: Agents course + práctica real (gratis, 5h + práctica)
```

Total: ~20h en 2 meses. Salís con baseline sólido para trabajar profesionalmente con Claude.

## Errores comunes al hacer los cursos

### 1. Pasar de uno a otro sin practicar

Tentación: "voy a hacer los 10 primero, después practico". **Error**. Hacé 1, practicá 1 semana, hacé 2, practicá. La retención sin práctica es ~10%.

### 2. Saltarse Claude 101 "porque ya uso Claude"

Sorprende cuánto cubre 101 que **no sabías**. Aunque uses Claude hace meses, vale el repaso de fundamentos.

### 3. Hacer cursos en lugar de proyectos

Hacer 8 cursos sin shippear un proyecto = waste. **2 cursos + 1 proyecto real** > 6 cursos sin práctica.

### 4. Ignorar los ejercicios

Los cursos vienen con hands-on. Si los skippeás "para terminar rápido", **el 70% del valor se pierde**.

### 5. No bajar el certificado

Si el curso lo ofrece, **descargalo**. Es señal de proactividad real en CV/LinkedIn. No es título universitario pero suma.
