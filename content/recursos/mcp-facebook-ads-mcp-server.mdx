## Qué es

Si manejás campañas en Facebook + Instagram (la suite de Meta Ads), este MCP te abre la API completa desde Claude Code. En vez de cambiar de pestaña al Ads Manager para chequear cómo va una campaña, modificar budgets o lanzar variantes, Claude lo hace por vos.

Cubre lo que la API de Meta Ads expone: campaigns, ad sets, ads, audiences, insights de performance, custom audiences, conversion tracking.

## Cómo se instala

```bash
npx claude-code-templates@latest --mcp marketing/facebook-ads-mcp-server
```

Pre-requisitos:
- Acceso a Meta Business Manager
- App registrada en [Meta for Developers](https://developers.facebook.com)
- Access token con scopes de Ads management

Para Claude:

```
Instalé el MCP marketing/facebook-ads-mcp-server. Después de configurar
las credenciales, listame las campañas activas en mi ad account, con
spend del día y CPM actual.
```

## Qué podés hacer con esto

### Auditoría de campañas en vivo

"Mostrame todas las campañas activas con CPM más alto del promedio del último mes, agrupadas por objective. Para cada una, decime si conviene apagar o ajustar budget".

### Creación de campaigns desde brief

"Creame una campaign Conversion en mi cuenta X, con ad set targeting [criterios] y 3 ads con estos copies + creatives". Claude estructura los API calls.

### Analytics combinado con tu data

"Cruzá el spend de campaña X con las conversions reales del CRM (no las que Meta reporta). ¿Cuál es el CPA real?". Útil cuando attribution se va de control.

### A/B testing programático

"Generá 5 variantes de copy para el ad `<id>`, lanzalas con budget igual y monitoreame CPC al cabo de 48 horas". Workflow de optimización iterativa.

### Audience management

"Creame una custom audience desde mis 1,000 emails de clientes top, después armame una lookalike 1% y prepará una campaña de prospección para esa audience".

### Pause masivo de campañas

"Pauseme todas las ads con CTR menor a 0.5% que llevan más de 48 horas activas". Higiene de cuenta.

## Pre-requisitos

- **Business Manager** con acceso a la ad account
- **App** en Meta for Developers con permisos Ads Management
- **Access token** con scopes correctos (NO expirado)
- **Knowledge de Meta Ads:** el MCP es plomería; las decisiones siguen siendo tuyas

## Cuándo NO conviene

<Callout type="danger" title="Antes de conectar Claude a presupuestos reales">

- **Pause / increase budget por error:** un prompt mal interpretado y Claude puede activar gasto inesperado. Para budgets serios, hooks de aprobación SIEMPRE.
- **Tokens y seguridad:** los access tokens de Meta Ads dan poder completo sobre la cuenta. Tratalos como credenciales bancarias.
- **Compliance:** crear ads con copy/creative que viole políticas de Meta puede bannear tu Business Manager. El MCP no valida políticas; vos sí.
- **Rate limits del API:** Meta Ads tiene cuotas. Operaciones masivas las podés bloquear.
- **Si solo necesitás reporting:** Meta Ads Manager o Looker conectado puede ser mejor UX.

</Callout>
