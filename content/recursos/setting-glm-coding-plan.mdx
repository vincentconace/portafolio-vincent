## Qué es

Setting de partnership que reconfigura Claude Code para usar **Z.AI GLM models** vía un endpoint compatible con la API surface de Anthropic. Es decir: misma client tooling, modelo distinto debajo.

Configura:
- `ANTHROPIC_BASE_URL: https://api.z.ai/api/anthropic`
- `ANTHROPIC_DEFAULT_HAIKU_MODEL: glm-4.5-air`
- `ANTHROPIC_DEFAULT_SONNET_MODEL: glm-4.7`
- `ANTHROPIC_DEFAULT_OPUS_MODEL: glm-4.7`
- `alwaysThinkingEnabled: true`

Pitch: GLM-4.6 es top-ranked en Code Arena leaderboard. Plan desde $3/mes (mucho más barato que Claude Pro).

## Cómo se instala

```bash
npx claude-code-templates@latest --setting partnerships/glm-coding-plan
```

Pre-requisitos:
- Cuenta Z.AI con GLM Coding Plan activo
- API key

Reemplazá `YOUR-API-KEY` con tu key real:

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "tu-key-real"
  }
}
```

Para Claude:

```
Instalé el setting partnerships/glm-coding-plan. Configurá la API key
de Z.AI. Hacé un test query y confirmá que el modelo que responde es
GLM-4.7 no Claude.
```

## Qué podés hacer con esto

### Reducción brutal de costo

Plan Z.AI desde $3/mes vs Claude Pro $20+. Para uso intensivo, diferencia significativa.

### Mismo workflow Claude Code

Como apunta a endpoint compatible Anthropic, no tenés que aprender nuevo client.

### Modelo competitivo en coding

GLM-4.6/4.7 puntea alto en Code Arena. Para tareas de coding puro, comparable a Claude.

### Combo con Z.AI MCPs

Si ya usás [Z.AI Vision MCP](/boveda/mcp-zai-mcp-server), [Web Reader](/boveda/mcp-web-reader), etc., centralizar en Z.AI ecosystem hace sense.

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **Si necesitás Claude específico:** algunos workflows fueron tuneados para Claude. Cambiar modelo puede degradar resultados.
- **Compliance:** Z.AI es empresa china. Algunas orgs tienen restricciones para mandar code a servicios chinos.
- **Performance variable:** modelos rotan en leaderboards. Verificá benchmarks recientes vs tu uso.
- **API compatibility no es 100%:** algunas features Anthropic-specific pueden no estar disponibles.
- **Data sensible:** tu code va a servidores Z.AI. Para projects regulados, validá policy.

</Callout>
