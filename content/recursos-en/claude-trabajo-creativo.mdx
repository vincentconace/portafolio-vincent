## The thesis

> **AI should expand what you already know how to do — not force you to learn new tools.**

Instead of launching "Photoshop with AI" or "Ableton with AI", Anthropic connected Claude **inside** the tools creatives already use. 8 official connectors built on MCP (the open protocol):

| Connector | Category |
|----------|-----------|
| **Adobe Creative Cloud** (50+ apps) | Design / video / web |
| **Affinity by Canva** | Design |
| **Blender** | 3D |
| **Autodesk Fusion** | CAD / parametric 3D |
| **Ableton Live & Push** | Music production |
| **Splice** | Music sample library |
| **SketchUp** | Architectural 3D modeling |
| **Resolume Arena & Wire** | Live VJing / projections |

On top of that: **Claude Design**, a new product for exploring visual ideas that exports directly to Canva.

---

## How the connectors work

Each connector is an **MCP server** that exposes tools specific to the app. Claude doesn't "replace" Photoshop or "control" Ableton — it **invokes** them with scoped tools.

For example, in Adobe:

```
Claude has tools like:
- list_open_documents
- get_layers(document_id)
- create_text_layer(content, position, font, color)
- export_as(format)
```

You talk naturally with Claude; Claude uses the tools to make real changes in your Photoshop / Illustrator / Premiere file.

**Advantage vs a traditional plugin**: Claude has context. It knows you're working on a campaign, it knows your brand guidelines (if you loaded them), it understands the brief.

## Connector by connector

### Adobe Creative Cloud

50+ apps covered. The most used ones:

#### Photoshop
- Generate text / shape layers with precise specs
- Suggest compositions
- Apply filters with calculated parameters (not just a "preset")
- Export variants in batch

```
> I have this design open. Generate 4 variants for an A/B test:
  - Version A: shorter headline
  - Version B: bigger CTA
  - Version C: layout with the image at the bottom instead of the top
  - Version D: combination of B + C

  Export them to /tmp/variants/ as PNG @ 2x.
```

#### Premiere Pro
- Automatic markers based on transcripts
- Cuts proposed according to audio rhythm
- Captions with precise timing
- Color grading suggested per scene

#### Illustrator
- Logo variants in different proportions
- Consistent icon systems
- Exports in every format / size you need

### Affinity by Canva

Similar to Adobe but in the Affinity ecosystem. Useful if your team is already there.

### Blender (3D)

- Modifiers applied to specific meshes
- Materials generated by description
- Basic animations (calculated keyframes)
- Render setup with professional presets

```
> I have this product model. Set up a studio scene for me to
  render in soft white/gray, 3 lights (key + fill + rim), camera
  slightly from above, 4K PNG output.
```

Anthropic also joined as a **sponsor of the Blender Development Fund** — the connector is part of a deeper relationship with the project.

### Autodesk Fusion (CAD)

Parametric modeling. Claude proposes:

- Sketches with dimensions
- Operations (extrude, revolve, sweep)
- Repetition patterns

Useful for industrial design, mechanical engineering.

### Ableton Live & Push

Music:

- Suggestions for harmonic progressions
- Stylistic drum patterns (not "random drumkit", but "Detroit techno 124bpm")
- Parameter automations
- Initial mix (proposed levels, EQ per instrument)

```
> I'm building a house track. I've got the drum groove done.
  Propose:
  - Chord progression in Am, 4 bars
  - Complementary bassline
  - Washed-out pad for drops

  Insert it on separate tracks so I can edit it.
```

### Splice

Smart sample search:

```
> I need a drum break that fits with what I have in the session.
  Search Splice for samples at 124 BPM, key A minor, "dirty/raw" vibe.
```

Claude searches Splice, brings you 5 options that match, and you can listen to them before downloading.

### SketchUp

Architectural modeling. Especially useful for:

- Generating quick variants of a layout
- Applying materials with judgment
- Setting up cameras for render

### Resolume (VJing / live visuals)

For visual performers:

- Setting up compositions from a description
- Mapping complex projections
- Automatic triggers based on incoming audio

## Claude Design — the new product

In addition to the 8 connectors, Anthropic released **Claude Design**: an Anthropic Labs product for **exploring visual ideas**.

It's not Photoshop. It's closer to **Midjourney + Figma combined**, where:

- You iterate visually on concepts
- Claude understands intent ("more editorial", "more brutalist", "more warm")
- Once you have something, you export **directly to Canva** to finalize it

**When to use it**:
- Exploring aesthetic direction early on
- Visual brainstorming before committing to a final piece
- Generating variants of a concept

**When NOT to**:
- Polished final work (that goes in Adobe / Affinity)
- Pieces with exact specs (final logos, official brand assets)

## Use cases by field

### Graphic design

Typical pack: Photoshop + Illustrator + Claude Design.

Flow:
1. Brief in Claude
2. Claude Design → conceptual variants
3. You take the one you like best to Photoshop/Illustrator
4. Claude operates inside for final iterations

### Music production

Pack: Ableton + Splice.

Flow:
1. Initial idea (groove, mood, genre)
2. Claude proposes progression + bassline in Ableton
3. Claude searches for complementary samples in Splice
4. You do the mix and final production

### 3D / Product

Pack: Blender or Fusion.

Flow:
1. You modeled the product (you)
2. Claude sets up the scene, lights, materials
3. Generates multiple render angles / variants
4. You pick the final ones

### Architecture

Pack: SketchUp.

Flow:
1. Initial layout (you)
2. Claude proposes variants (change circulation, materials)
3. Comparative renders
4. You decide

### Live performance / VJ

Pack: Resolume.

Flow:
1. You set up the base session
2. Claude configures mappings, transitions, triggers
3. During the show, Claude can respond to audio changes in real time

## What the connectors do NOT do

Important so you don't expect too much:

- **They do NOT create finished work for you**. They assist you in specific steps. The vision is still yours.
- **They do NOT understand your brand without context**. If you want them to respect your brand guidelines, hand them over explicitly.
- **They do NOT replace technical skills**. If you don't know Photoshop, the connectors won't teach you how to use it — they'll help you be faster at what you already know.
- **Some advanced features require a high-tier plan**. Check what your Claude subscription includes.

## Anti-patterns

### 1. Expecting AI to "have taste" instead of judgment

Claude helps you iterate faster. It doesn't give you taste. If you don't have a clear vision, generating 50 variants won't save you — it'll confuse you more.

### 2. Skipping fundamentals

If you've never opened Photoshop, the connectors won't make you a designer. Learn the craft; use AI as an amplifier.

### 3. Total dependence

If your entire workflow depends on the connectors and one day they're down / they change the API, you're left with no way to work. **Keep the ability to operate manually** as a fallback.

### 4. Sharing heavy credentials

The connectors ask for access to your Adobe / Ableton / etc. account. If you work at a company, **respect IT policies** — some connectors may not meet compliance.

## How to get started

1. Identify the 1-2 creative tools you use the most
2. Enable only those connectors (not all of them)
3. Spend 1 week using them on real tasks
4. Decide whether they're worth it for your specific workflow

Don't enable all 8 on day 1. It's overhead you probably don't need.
