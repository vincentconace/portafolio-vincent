## Herramienta 1 — Awesome Claude Skills

### Analogía

Claude es como un chef que sabe cocinar, pero **sin recetas**. Awesome Claude Skills es el libro de recetas. Le das el libro y de repente sabe hacer hooks que detienen el scroll.

### Qué hace

- Habilidades listas para escribir contenido viral
- Hooks que detienen el scroll
- Estructuras de video probadas
- Patrón de "hook + body + payoff + CTA"

Repo: `ComposioHQ/awesome-claude-skills`. Open source.

### Instalar

```
> Entrá a este repositorio:
  https://github.com/ComposioHQ/awesome-claude-skills

  Buscá las habilidades de contenido viral y escritura de guiones.
  Instalá la que sirva para escribir hooks y estructurar videos
  virales.

  Usá esa habilidad para todo lo que te pida de ahora en adelante.
```

## Herramienta 2 — UltraThink

### Analogía

Cuando le pedís algo, Claude **responde rápido** pero no se detiene a pensarlo mucho. Como preguntarle a un amigo mientras camina — te da respuesta rápida.

UltraThink es como decirle: **"Sentate y pensalo bien antes de contestar"**.

### Qué hace

- Razonamiento profundo
- Compara diferentes ángulos
- La diferencia entre guion aburrido y bien estructurado

### Cómo se usa

Una sola palabra al inicio del prompt:

```
ultrathink — escribime guion viral para [TEMA] en formato 45s
para TikTok.
```

**No necesitás instalar nada**.

## Herramienta 3 — VoxCPM (clonación de voz)

### Qué hace

- Cloná tu voz **en segundos**
- 30 idiomas soportados
- Calidad de estudio
- Gratis, open source

### Setup

```
> Instalá VoxCPM en mi computadora.

  Para clonar mi voz:
  1. Tomá audio de muestra (30s de mi voz hablando claro)
  2. Procesalo
  3. Generá mi clon vocal
  4. Probalo con frase de prueba
```

### Usar

```
> Tomá el guion que generamos y usá VoxCPM con mi voz clonada
  para generar el audio del Reel.

  Tono: enérgico al principio, calmado en payoff.
  Output: /audio/reel-[FECHA].mp3
```

## El flow completo

### Brief (30 segundos)

```
> ultrathink — necesito Reel de 45s sobre [TEMA].

  Audiencia: [TARGET]
  Objetivo: [educar / entretener / vender]
  Plataforma: [TikTok / Reels / Shorts]

  Usá Awesome Claude Skills para estructura viral.

  Generá:
  1. Hook (3 seg primeros, debe detener scroll)
  2. Body (3 puntos listicle)
  3. Payoff (revelación / conclusión)
  4. CTA

  Después de guion, generá audio con VoxCPM usando mi voz clonada.
```

### Output (~2-3 minutos)

- Guion 45-55s
- Audio MP3 con tu voz

### Ensamblar en video

Pegás el audio en CapCut o [Editor Pro Max](/boveda/editor-pro-max) con visuales. Total: **10-15 min para Reel publicable**.

## Anti-patrones

### 1. Usar UltraThink para todo

`ultrathink + dame fecha` = overhead. Solo para razonamiento que realmente lo justifique.

### 2. Esperar audio perfecto en first try

VoxCPM es bueno pero no perfecto. **Itera**. Si el tono no convence, regenerá con instrucción más específica.

### 3. No humanizar el guion

Aun con Awesome Claude Skills, el output puede sonar IA. **Pasalo por [humanizalo](/boveda/humanizalo)** antes de generar audio.

### 4. Clonar tu voz con audio de mala calidad

Si tu sample de 30s tiene ruido de fondo, **el clon va a sonar raro**. Grabá clean.

### 5. Publicar sin revisar

Workflow es rápido pero **siempre revisá** antes de publicar. Errores de pronunciación, palabras raras, etc.

---

## Limitaciones éticas

### Clonación de voz

VoxCPM clona TU voz. Para clonar voz **de otra persona** necesitás permiso explícito. Caso contrario es **deepfake**.

### Disclosure

En algunas plataformas (TikTok, YouTube), si usás voz IA, **divulgalo**. Las reglas cambian — verificá.
