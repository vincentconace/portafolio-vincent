## The combination

Imagine you have:
- **A book with 67 design styles** (UI/UX Pro Max GO)
- **An artist who draws whatever you ask** (NanoBanana)
- **An interface designer who generates HTML** (Stitch)
- **A store of premium components** (21st Dev)

**That's the Ultimate Web Designer**. 4 tools you combine so that Claude:
- Knows design with professional judgment
- Generates images and mockups with AI
- Turns ideas into HTML/CSS
- Uses pre-built premium components

| Piece | Role |

## Setup — 4 pieces

### Piece 1 — UI/UX Pro Max GO

```bash
# Install globally
npm install -g uipro-cli

# Configure for Claude Code
uipro init --ai claude
```

Generates config automatically. Claude has **67 styles + 161 palettes + 57 fonts + 99 UX rules** available.

### Piece 2 — NanoBanana MCP

See [Restore your photos](/boveda/restaura-tus-fotos) for detailed setup.

Quick:
```
> Install NanoBanana MCP. My Gemini API key: [KEY]
```

### Piece 3 — Google Stitch MCP

See [Stitch AI design](/boveda/stitch-diseno-ia) for detailed setup.


### Piece 4 — 21st Dev Magic

```bash

```


## Complete workflow

### For a new landing page

```
> Design a landing page for [PRODUCT/SERVICE].

  Audience: [TARGET]
  Style: [MOOD - use UI/UX Pro Max to suggest 3 options from
  the 67 styles]

  Steps:
  1. UI/UX Pro Max: define palette, typography, style
  2. NanoBanana: generate hero and section images (3-5)
  3. Stitch: build the HTML/CSS structure of the sections
  4. 21st Dev Magic: bring in premium components for buttons,
     forms, cards
  5. Assemble everything into a Next.js + Tailwind project

  Deliver:
  - A complete, working landing page
  - List of 21st Dev components used
  - Generated images in /public/images/
```

20-40 min later: **a production landing page**.

### To improve an existing landing page

```
> Here's my current landing page: [path or URL]

  Improve it:
  1. UI/UX Pro Max: audit it. What's generic / what looks like a template
  2. NanoBanana: replace stock images with AI-generated ones
  3. 21st Dev Magic: replace default shadcn components with
     premium ones
  4. Stitch: adjust the layout if sections look flat

  Keep the copy and CTAs. Visual upgrade only.
```

## Anti-patterns

### 1. Installing 4 pieces and not using all of them

Each tool has a role. If you **only use UI/UX Pro Max**, you lose Stitch + NanoBanana + 21st Dev. **Learn the full flow**.

### 2. A vague brief

❌ "Make me a pretty landing page"
✅ "Landing page for a B2B SaaS inventory tool, audience LATAM COOs, Linear-inspired aesthetic, earthy green palette"

### 3. Expecting pixel-perfect on the first try

The stack is good, but **you iterate 2-3 times** before finalizing. Accept the iteration.

### 4. Overusing NanoBanana

Generating 30 images "just in case" burns credits. **Generate the ones you're going to use**. If in doubt, generate 1 version and test it.

### 5. Ignoring mobile

The stack generates responsive by default, but **test it on real mobile**. Sometimes you need to adjust.
