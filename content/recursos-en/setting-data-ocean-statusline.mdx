## What it is

The fourth gimmicky statusline in the "fun" family. Ocean theme:

- **Depth** based on the count of files you touch
- **Sea creatures** appearing based on context
- **Treasures** occasionally, as a reward

It's the least adopted of the 4 gimmicky ones (69 installs vs. 141 for Spaceship, 176 for Circus, 66 for Casino).

## How to install it

```bash
npx claude-code-templates@latest --setting statusline/data-ocean-statusline
```

For Claude:

```

current depth and which sea creatures I'm going to run into.
```
