## Qué es

Es uno de los marketplaces más grandes y mejor curados que vas a encontrar para Claude Code. Lo mantiene Alireza Rezvani y junta 192+ skills bajo 28 plugins temáticos. La gracia es que no es solo "engineering": tiene packs separados para marketing, product, project management, finanzas, compliance regulatoria (HealthTech/MedTech con ISO 13485, FDA, GDPR) y un pack de C-level con junta directiva virtual.

Funciona en Claude Code y otros 8 entornos de coding agent — si mañana cambiás de herramienta, las skills te siguen.

No instalás los 28 plugins a la vez (sería absurdo). Agregás el marketplace y después elegís 2-3 packs según el rol que estés jugando esa semana.

## Cómo se instala

```bash
# 1) Agregar el marketplace
/plugin marketplace add alirezarezvani/claude-skills

# 2) Instalar los packs según rol (no los instales todos)
/plugin install engineering-skills@alirezarezvani-claude-skills
/plugin install marketing-skills@alirezarezvani-claude-skills
```

O `/plugin` → Discover.

Para que Claude haga la curación según lo que necesitás:

```
Instalá el marketplace alirezarezvani/claude-skills. Soy [tu rol: dev full-stack
/ founder / PM / etc.] trabajando en [tu proyecto]. ¿Cuáles 2-3 packs de los 28
disponibles me convienen instalar primero y por qué?
```

## Qué podés hacer con esto

### Cubrir un rol nuevo en una semana

Si esta semana tenés que actuar como Project Manager (tu PM se fue), instalás `pm-skills` y de golpe Claude sabe cómo escribir tickets de Jira siguiendo la convención del pack, armar un sprint plan y facilitar una retro. No reemplaza a un PM senior, pero te tapa el hueco.

### Lanzar contenido sin equipo de marketing

`marketing-skills` viene con 43 skills agrupadas en 7 pods. Para un founder solo: pedirle a Claude que use el pack de Content + SEO + CRO te da un calendario editorial, una landing optimizada y un análisis de conversion en la misma sesión.

### Compliance pesada (HealthTech)

Si trabajás en algo regulado (`ra-qm-skills`), los 13 skills de regulatorio cubren ISO 13485, MDR EU, FDA 510(k), CAPA management. No te valida ante la FDA, pero te ahorra meses de research sobre qué exige cada framework.

### Stress-test de un plan con `executive-mentor`

Como [Adversarial Spec](/boveda/spec-kit), pero para decisiones de negocio: presentás tu plan y la skill lo critica con preguntas duras. Útil antes de un board meeting o una decisión irreversible.

### Junta virtual de C-level

`c-level-skills` activa "asesores" sintéticos (CEO, CTO, COO, CPO, CMO, CFO, CRO, CISO, CHRO) que te dan opiniones desde sus ángulos. No reemplazan a humanos reales, pero ayudan a estructurar el pensamiento.

## Beneficios reales

- **Cobertura horizontal real** — un solo marketplace cubre eng/marketing/product/finanzas. Pocos lo hacen.
- **Cross-tool** — el día que probás Cursor o Codex, no perdés las skills.
- **Mantenimiento activo** — 7.5k stars no son por casualidad. El autor itera.
- **Granular** — instalás solo los packs del rol que estás jugando esta semana.

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **No instales los 28 packs juntos:** vas a tener 192 skills cargadas y Claude se va a marear. Elegí 2-3 según rol.
- **Si ya pagás herramientas dedicadas** (Notion AI para marketing, Linear AI para PM), las skills se solapan. Probá una antes de duplicar.
- **`ra-qm-skills` no te certifica nada:** son guías y templates. La validación regulatoria sigue siendo humana.
- **`c-level-skills` no reemplaza mentores reales:** úsalo para estructurar ideas, no para decidir solo.

</Callout>
