## Qué es

`spinnerTipsOverride` te deja reemplazar los textos que aparecen rotando mientras Claude piensa. `excludeDefault: true` quita los defaults; tu array `tips` los reemplaza.

Casos: humor team-specific, reminders, brand voice.

## Tabla resumen

| Campo | Valor |
| --- | --- |
| Categoría aitmpl | `global` |
| Installs | 36 |
| Setting key | `spinnerTipsOverride` |

## Cómo se instala

```bash
npx claude-code-templates@latest --setting global/spinner-tips-override
```

Editá el array `tips` con tus mensajes.

Para Claude:

```
Instalé el setting global/spinner-tips-override. Mostrame el array
default y reemplazálo con [tips custom de mi equipo].
```

## Qué podés hacer con esto

### Brand voice constante

Tu org tiene tono específico. Reemplazás defaults con tus phrases.

### Reminders team

"Quedan 2 días para sprint review" como spinner tip.

### Humor team

Tips chistosas que solo tu equipo entiende.

### Combo con [Friday Deploy Warning](/boveda/setting-friday-deploy-warning)

Friday Deploy es un caso específico. Spinner Tips es más genérico.

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **Solo 36 installs:** muy nicho.
- **Si los defaults te alcanzan:** override es overhead.
- **Para uso individual:** less útil que team-shared.

</Callout>
