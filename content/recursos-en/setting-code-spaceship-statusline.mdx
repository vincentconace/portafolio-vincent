## What it is

Spaceship-themed statusline. The statusline shows:
- **Fuel** that gets consumed with every operation
- **Distance** you travel on your coding journey
- **Warp levels** based on work intensity

It's the most-adopted of the gimmicky family (141 vs 66/176/69 for the others).

## How to install it

```bash
npx claude-code-templates@latest --setting statusline/code-spaceship-statusline
```

For Claude:

```
I installed the statusline/code-spaceship-statusline setting. Show me the
current fuel level and warp speed.
```

## What you can do with it

### Gamifying your work

"I'm going to fill up the tank before doing this refactor." The framing can help with engagement.

### Sci-fi theme for fans

If you like sci-fi, the theme resonates with you more than other statuslines.

### The most-adopted "fun" one

Of the 4 gimmicky ones (Circus, Casino, Spaceship, Ocean), this has the most users. Maybe it's more balanced.

## When it's NOT worth it

<Callout type="warning" title="Before installing">

- **For serious work:** [Context Monitor](/boveda/setting-context-monitor) is the grown-up option.
- **If the spaceship theme doesn't interest you:** without emotional buy-in, it adds nothing.
- **For clients / presentations:** unprofessional.

</Callout>
