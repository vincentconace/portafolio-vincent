## What it is

TinyFish is a web agent platform focused on data extraction, with two distinctive features:
1. **Real-time progress streaming** — you know what the agent is doing while it runs, not just the final result
2. **Stealth browser profiles** — the browser blends in as real traffic, not as a bot

Built for extraction where other tools fail because of anti-bot defenses. Comparable to [Browserbase](/boveda/mcp-mcp-server-browserbase) but with more of a focus on extraction + streaming.


Prerequisite: a TinyFish account + API key.

For Claude:

```

products from an Amazon page. Launch the task async with a stealth

```

## What you can do with it

### Scraping with anti-bot

"Go to a page that has Cloudflare. With a stealth profile, navigate and extract the listings." Where a local browser fails, TinyFish can get through.

### Async tasks with progress

"Launch the scrape task over 200 pages. While it runs, show me progress every 30s. When it finishes, return all the unified results."

### Multi-page extraction

"Walk through all the pagination on site X and extract every item. Report back the total and any errors."

### Structured data extraction

"From this product page, extract `nombre`, `precio`, `descripción`, `specs`, `reviews count` as JSON."

### Compared to [Bright Data](/boveda/mcp-brightdata) / [Browserbase](/boveda/mcp-mcp-server-browserbase)

- **Bright Data:** enterprise scale, 60+ tools, more expensive
- **Browserbase:** cloud browser + Stagehand, more general-purpose
- **TinyFish:** focused on extraction + stealth, likely more accessible pricing

## When it's NOT worth it

<Callout type="warning" title="Before installing">

- **For sites without anti-bot:** [Firecrawl](/boveda/mcp-firecrawl) or your own script is simpler.
- **Costs:** stealth profiles + cloud browsers charge. Check the pricing.
- **Legal:** evading bot detection may violate the TOS. The legal responsibility is yours.
- **Only 81 installs:** a small community trying it out.
- **For a one-off scrape:** adding another service may be overkill.

</Callout>
