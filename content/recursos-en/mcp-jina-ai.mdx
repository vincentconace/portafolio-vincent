## What it is

Jina AI is one of the companies pushing hardest on infrastructure for LLM-native apps: embeddings, reranker, web reading, grounding, fact-checking. This official MCP integrates several of its APIs into Claude Code.

The key capabilities:
- **Web reading** — convert URLs into clean markdown (similar to [Firecrawl](/boveda/mcp-firecrawl))
- **Web search** — searches with structured results
- **Grounding** — attach real citations to the claims Claude makes
- **Fact-checking** — validate claims against web sources

The most distinctive use is **grounding**: anti-hallucination with verifiable citations. When you care about how much you can trust the answer (research, public content, technical documentation), the MCP turns "Claude said X" into "X, with this source you can verify".

## How to install it

```bash
npx claude-code-templates@latest --mcp devtools/jina-ai
```

Prerequisite: a Jina AI API key.

For Claude:

```
I installed the devtools/jina-ai MCP. Configure the API key and show me
which tools it exposes: web reading, search, grounding, fact-checking. Then
run a search with grounding about [topic] and return the claims
with their citations.
```

## What you can do with this

### Reduce hallucination with grounding

"Answer me with grounding enabled: what are the newest features in Postgres 17?". Claude answers citing verifiable sources, not from memory.

### Fact-check outputs

"For this blog post draft, fact-check every claim. Flag the ones you can't validate". Before publishing, validation.

### Research with citations

"Research topic X and return a 5-point summary, each one with its source". Useful for research that later gets published.

### Web reading with LLM-ready quality

Like [Firecrawl](/boveda/mcp-firecrawl), Jina converts URLs into clean markdown. Compare them based on your preference.

### Anti-hallucination pipeline

Combined with [Context7](/boveda/mcp-context7) for libraries and Jina for general info on the web, you cover two big vectors of hallucination.

## Prerequisites

- **Jina AI account** with API key
- **Internet connection** (everything goes through their APIs)

## When it's NOT a good fit

<Callout type="warning" title="Before installing">

- **For offline or private info:** Jina indexes the public web. Your internal wiki doesn't apply.
- **Costs:** some APIs charge per use. Check pricing before batching.
- **Only 17 installs:** limited community testing. Try it in a sandbox first.
- **If you already use [Firecrawl](/boveda/mcp-firecrawl)** for web reading: there's overlap. Try one before adopting both.
- **For grounding over your private data:** Jina doesn't reach it. Look at your own RAG.

</Callout>
