## What it is

Footballbin Predictions is a service that provides AI forecasts for football matches — specifically the English Premier League and the Champions League. The MCP plugs that service into Claude Code. You ask it for predictions, and it returns:

- Half-time score
- Full-time score
- Next goalscorer
- Number of corners

It's niche. If you build fantasy football apps, football content, or agents that track matches, it saves you from writing the API client.

## How to install

```bash
npx claude-code-templates@latest --mcp integration/footballbin-predictions
```

Prerequisite: a Footballbin account and API key (if applicable).

For Claude:

```
I installed the integration/footballbin-predictions MCP. I'll ask for the prediction
of Real Madrid's next Champions League match: half-time score,
full-time score, next goalscorer and corners.
```

## What you can do with it

### Automated content

"Generate an Instagram post with the prediction for today's match X: a visual of the expected score, estimated top scorer, corner count." A content pipeline for football accounts.

### Fantasy apps

If your fantasy app needs estimated scores or probabilities, the MCP provides the feed. Useful for suggesting to users which players to pick up.

### Comparative analysis

"Compare the MCP's predictions vs the actual results of Liverpool's last 10 matches. What's the accuracy?" Useful for validating before using in production.

### Themed bots / agents

An agent that tweets predictions before each match, contrasts them with the result, and builds a weekly accuracy score.

## Prerequisites

- **Active Footballbin account**
- **API key** if applicable
- **Focus on Premier League / Champions League:** other leagues are not covered

## When it's NOT a good fit

<Callout type="danger" title="Before using it to bet money">

- **These are NOT betting recommendations.** They're AI predictions. They can be wrong. Use them for entertainment, analysis or content — not for throwing money around without your own judgment.
- **Limited coverage:** only Premier League + Champions League. If your interest is La Liga, Bundesliga, Serie A, NBA, NFL, etc., this MCP doesn't apply.
- **Unknown accuracy:** there's no public benchmark. Validate against history before trusting it.
- **Betting and legality:** betting is regulated by country. You are responsible.
- **Only 51 installs:** small community. Test it in a sandbox first.

</Callout>
