## What it is

Trello is the most popular Kanban board. Boards, lists, cards. The inline description on aitmpl is generic — for details, see the [configuration JSON](https://github.com/davila7/claude-code-templates/blob/main/cli-tool/components/mcps/devtools/mcp-server-trello.json) and the project's official docs.

This MCP connects to your Trello account so Claude can operate boards and cards from the editor. Handy when you organize work in Trello but code in Claude.

## How to install

```bash
npx claude-code-templates@latest --mcp devtools/mcp-server-trello
```

Prerequisite: a Trello account + API key + token from [trello.com/app-key](https://trello.com/app-key).

For Claude:

```
I installed the devtools/mcp-server-trello MCP. Set up the API key + token and
list my boards. Show me the cards in the "Doing" state of board X.
```

## What you can do with this

### Task triage

"In board X, list the oldest cards that haven't moved in 14 days. For each one, tell me whether it needs to be revived, archived, or bumped in priority."

### Sync with code

"For every PR I closed today, move the related card to `Done` in board X." A post-merge update pipeline.

### Creating cards from code

"From the issues I closed this sprint, create retro cards for me: lessons learned and action items for the next sprint."

### Cross-board search

"Across all my boards, find cards with the `urgent` label that haven't moved in the last week."

### Team notifications

Combined with [Slack or others](/boveda/mcp-google-workspace), when a critical card moves to Done, Claude can notify the corresponding channel.

## Prerequisites

- **Active Trello account**
- **API key + token** from the dev panel
- **Permissions** on the boards you're going to manipulate

## When it's NOT worth it

<Callout type="warning" title="Before you install">

- **If you use another management platform** ([monday.com](/boveda/mcp-monday), Asana, Notion, ClickUp): this MCP is Trello-only.
- **Destructive operations:** archiving / deleting cards by mistake. Use approval hooks.
- **Broad token:** a token lets Claude operate all your visible boards. Treat it as a serious credential.
- **Already comfortable manual workflow:** if your interaction with Trello is casual, the MCP adds overhead.

</Callout>
