## What it is

**Antigravity Awesome Skills** is a repository with **978+ markdown instruction files**. Each file teaches your AI agent how to do a specific task.

Think of it this way: each skill is like **hiring an expert consultant on demand**. Your agent reads them automatically and uses them when they're relevant to what you're doing.

Repo: `sickn33/antigravity-awesome-skills`. ~32k stars.

## Installation

Open a terminal in your project folder and run the command for your tool:

### Claude Code

```bash
npx antigravity-awesome-skills --claude
```

### Cursor

```bash
npx antigravity-awesome-skills --cursor
```

### Gemini CLI

```bash
npx antigravity-awesome-skills --gemini
```

### Codex

```bash
npx antigravity-awesome-skills --codex
```

### Other compatible agents

```bash
npx antigravity-awesome-skills
```

The installer lets you **choose which categories to install**. **Do NOT install all 900+**. Pick 2-3 categories relevant to your project.

## How to use a skill

Once installed, you use the `@skill-name` syntax to activate a skill in your prompt:

### Examples

```
@api-design design the API for my ordering app
```

```
@security-audit review this endpoint for vulnerabilities
```

```
@pricing-strategy help me set prices for my SaaS
```

```
@test-generation generate tests for this module
```

```
@refactor-suggestion look at this code and suggest a refactor without
breaking functionality
```

## Recommended adoption strategy

### Day 1 — Install 2 core categories

The 2 that match **your main work**. No more.

### Week 1 — Try 5-10 skills

Identify which ones you actually use. Most won't apply to you.

### Week 2 — Refine

Keep the **20-30 skills** you use most. Uninstall the rest (they take up context).

### Month 2 — Secondary categories

If your workflow expands, add another category.

### Month 3+ — Create your own

The Antigravity skills are a baseline. For your specific workflow, **create your own** ([create skills](/boveda/crea-claude-skills)).

## The good and the not-so-good

### Pros

- ✅ **Massive quantity** — 978+ is hard to beat
- ✅ **Open source and maintained** — active community
- ✅ **Multi-agent** — works with all the major ones
- ✅ **Well-organized categories** — easy to filter
- ✅ **Trivial setup** — a `npx` one-liner

### Honest cons

- ⚠️ **Variable quality** — some skills are great, others are basic
- ⚠️ **Choice paralysis** — 978 options can be paralyzing
- ⚠️ **Genericness** — being built for "everyone," it doesn't fit your specific niche
- ⚠️ **Doesn't replace custom skills** — your own workflow needs your own skills

## Anti-patterns

### 1. Installing all 9 categories "just in case"

Every skill takes up context. With 978 active, **Claude gets distracted** trying to find a match. Install 2-3 relevant categories.

### 2. Not searching before asking

Before asking for something, **list the available skills**:

```bash
npx antigravity-awesome-skills --list
```

Sometimes what you need already exists. Asking Claude without invoking the right skill = you're not making the most of Antigravity.

### 3. Forgetting to remove

If you installed a category and don't use it, **remove it**. Every category adds weight to the context.

### 4. Creating your own skill when Antigravity already has it

Before building your own `@deploy-vercel`, **check** whether Antigravity has `@vercel-deployment`. If it already exists, save yourself the setup.

### 5. Expecting magic from a vague prompt

`@security-audit do your thing` → mediocre output.

`@security-audit review my /api/orders endpoint for:
- SQL injection
- IDOR
- Auth bypass
- Rate limiting
Report with severity and a suggested fix` → useful output.

**The skill activates the knowledge. The prompt activates the format and depth**.

## When it's NOT worth it

| Case | Why |
|------|---------|
| You want uniformly high quality | Antigravity has variance |
| Your niche is very specific | Generic skills don't fit |
| You prefer manual curation | Too much quantity |
| Your workflow already has mature custom skills | Unnecessary overlap |

If you have your own set of proven skills, Antigravity is **complementary**, not central.
