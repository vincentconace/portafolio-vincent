## Qué es

Le decís a tu compu: **"hacéme video de 30s anunciando mi producto"** y ella lo hace sola.

**Editor Pro Max** es una herramienta open source que convierte Claude Code en estudio de edición de video con IA. Describís en español → Claude escribe código Remotion → exportás MP4 profesional para cualquier plataforma.

Repo: `Hainrixz/editor-pro-max`. Open source.

| Característica | Detalle |
|----------------|---------|
| Procesamiento | 100% local (nada sube a cloud) |
| Stack | Remotion 4 + FFmpeg + Whisper.cpp + React 19 |
| Componentes | 25+ |
| Templates | 9 (TikTok, Reels, YouTube, etc.) |
| Plataformas output | TikTok, IG, YouTube, LinkedIn, Twitter, FB |
| Costo | Gratis |

---

## Pre-requisitos

- **Claude Code** instalado
- **Node.js 20+**
- **FFmpeg** (opcional, para procesar video existente)
- **Whisper.cpp** (opcional, para subtítulos automáticos)

---

## Setup — 4 comandos

```bash
# 1. Clonar
git clone https://github.com/Hainrixz/editor-pro-max.git

# 2. Instalar
cd editor-pro-max && npm install

# 3. Abrir Claude
claude

# 4. Iniciar preview
npm run dev
```

Abrí `http://localhost:3000` y vas a ver el **Remotion Studio** con preview en vivo.

### FFmpeg y Whisper son opcionales

El core de creación funciona sin ellas. Solo las necesitás para:
- FFmpeg → procesar video existente (cortar silencios, jump cuts)
- Whisper → subtítulos automáticos

## 4 prompts para empezar

### 1. TikTok desde cero

```
> Creá TikTok de 30s para promocionar [mi producto/servicio].
  Estilo: moderno y dinámico.
  Incluí:
  - Texto animado con nombre del producto
  - 3 beneficios principales con iconos
  - CTA final
  Colores: [colores de mi marca]
  Formato 9:16 TikTok.
```

### 2. Subtítulos a video existente

```
> Tengo video en [ruta/al/video.mp4].
  Agregá subtítulos automáticos con preset "bold" (grandes,
  alto contraste).
  Transcribí con Whisper y sincronizá con el habla.
```

### 3. Presentación profesional

```
> Creá presentación de 2 min sobre [tema].
  Formato 16:9 para YouTube.

  Estructura:
  - Intro con título animado
  - 5 slides con puntos clave
  - Transición suave entre slides
  - Cierre con logo y contacto

  Estilo: minimalista, fondo oscuro, texto blanco con acentos
  [color de marca].
```

### 4. Limpiar video largo

```
> Tengo video largo en [ruta/al/video.mp4]. Necesito:

  1. Detectar y eliminar silencios mayores a 0.5 segundos
  2. Aplicar jump cuts automáticos para dejar solo el habla
  3. Exportar como MP4 para YouTube (16:9)
```

## Anti-patrones

### 1. Prompts vagos para video

❌ "Hacéme un video bonito"
✅ "TikTok 30s, estilo Linear, dark mode, 3 beneficios con icons + CTA"

Especificidad gana.

### 2. Una sola iteración

Primera versión es 70%. Iterá:
- "El texto del segundo 5 sale muy rápido, ralentizá"
- "Cambiá el color del CTA a verde"
- "Agregá pause de 0.5s antes del logo"

### 3. Esperar pixel-perfect creative

Editor Pro Max es **excelente para video programático**. Para edición creativa con muchas tomas (filmadas, juegos visuales sutiles), After Effects sigue ganando.

### 4. Ignorar el preview

`localhost:3000` te muestra el video **antes de render**. Render toma minutos. **Iterá en preview**, render solo cuando estás listo.

### 5. Render en mala calidad por ahorrar

Si vas a publicar en YouTube/Reels, **renderizá en máxima calidad**. Los segundos extra valen.
