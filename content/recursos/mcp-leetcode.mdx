## Qué es

LeetCode es la plataforma estándar para prep de entrevistas técnicas. 3,000+ problemas, sistema de difficulty, contests, soluciones de usuarios.

Este MCP da a Claude Code acceso al API de LeetCode: buscar problemas, ver enunciados, traer info del usuario, contest data. Útil si:
- Estás prepando entrevistas y querés que Claude te asista
- Construís herramientas de tracking de progreso
- Querés un workflow tipo "Claude me elige un problema random del topic X y me asiste con hints"

## Cómo se instala

```bash
npx claude-code-templates@latest --mcp devtools/leetcode
```

Para Claude:

```
Instalé el MCP devtools/leetcode. Mostrame el problema "two-sum" con
su enunciado completo y constraints. Después dame 3 enfoques distintos
de solución, del más naive al más optimizado, con análisis de
complexity.
```

## Qué podés hacer con esto

### Estudio asistido

"Buscame un problema de difficulty Medium, topic Dynamic Programming, que no haya resuelto. Dame el enunciado y dejá que yo intente. Después podés revisar mi solución".

### Plan de estudio

"Generame un plan de 2 semanas para entrevistas senior backend: prioridad a `graph traversal`, `system design`, `concurrency`. 2-3 problemas por día, mix de Medium y Hard".

### Tracking de progreso

"De los problemas que resolví este mes, ¿cuáles topics me cuestan más (más intentos antes de pasar)? Sugerime los topics a reforzar".

### Análisis de contest

"Mostrame el contest más reciente. Para cada problema, dame mi tiempo vs el del top 1%. Sugerime qué patrón estudiar para subir".

### Soluciones explicadas

"Para el problema X, mostrame mi solución y compárala con la official solution. Explicame las diferencias en complexity y por qué la oficial es mejor".

### Mock interview asistido

"Hagamos mock interview: dame un problema Hard, dame 5 minutos para entender, después yo cuento mi enfoque y vos me hacés preguntas como entrevistador". Workflow conversacional.

## Pre-requisitos

- **Cuenta LeetCode** si vas a usar features que requieren login (tu progress, contest results)
- **Conexión a internet** (todo es API contra LeetCode)

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **Si no estás prepando entrevistas:** el MCP es nicho. Para problemas algorítmicos genéricos podés pedirle a Claude directo.
- **Para cheating en interviews live:** lo dejamos claro — está mal y se nota. Usalo para estudio honesto.
- **Para problemas patentados de la empresa:** LeetCode no tiene todos los problemas de cada empresa. Algunos son customizados.
- **Si tu prep es de system design:** LeetCode tiene poca cobertura ahí. Mirá otros recursos.

</Callout>
