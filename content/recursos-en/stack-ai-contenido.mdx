## The problem

Every week a new tool comes out. You try 3, keep 2, drop 1, move on to the next. Result: **5 tabs open, 3 subscriptions you don't use, Friday night** trying to remember which one was the good one.

**A 1-min Reel takes you 4 hours** because you jump between Claude, Gemini, Midjourney, Runway, Descript, CapCut, Canva, Notion. Every jump = copy-paste, re-prompt, new account. By the time you post it, the idea has already passed.

**The solution**: a small, fixed stack, where each tool does **one specific thing** and they all connect through **a single brain on top** — Claude for text/direction + a Master Prompt that generates perfect prompts for each model.

| Before | After |
|-------|---------|
| 15 tabs · 6 accounts · 0 stack | 6 tools · 1 Master Prompt |
| You re-pick the tool at every step | Each tool has a fixed role |
| **4 hours per Reel** | **40 min per Reel** |

## The map — 6 links in a chain

### 1. Script → Claude (Opus 4.7 + Sonnet)

Turns a raw idea into a **humanized 45–55s script** with hook + listicle body + payoff + CTA.

- **Opus 4.7** for structural reasoning (how to chain arguments together)
- **Sonnet** for fast iteration on variants

### 2. Image → Nano Banana Pro + ChatGPT Image

- **Nano Banana Pro**: speed for drafts and static B-roll
- **ChatGPT Image**: photoreal detail for covers

### 3. Video → Veo 3.1 + SeeDance 2

- **Veo 3.1**: cinematic realism
- **SeeDance 2**: speed for iteration

What used to be a full day of shooting, **10 minutes**.

### 4. Voice → ElevenLabs

- Voice cloning in **30 seconds**
- Character voices for narration, dubbing, brand ASMR

### 5. Avatar → HeyGen + Higgsfield

- **HeyGen**: your UGC avatar talking to camera
- **Higgsfield**: cinematic image-to-video effects

See [Higgsfield MCP](/boveda/higgsfield-mcp) for direct integration.

### 6. DMs → ManyChat

Automates replies to comments and DMs. If your Reel goes viral, ManyChat **turns your audience into leads**.

## Real workflow (40 min)

### 0-5 min — Brief

You throw Claude the idea + audience + format.

### 5-15 min — Master Prompt

Claude generates:
- A humanized script (runs through the humanizer skill)
- Prompts for each tool in the stack

You validate. You adjust 1-2 things.

### 15-25 min — Visual assets

While Claude is thinking, in parallel:
- Nano Banana → cover
- Veo 3.1 → 3 B-roll clips
- HeyGen → your avatar talking

### 25-30 min — Voice + audio

ElevenLabs reads the script with your cloned voice.

### 30-35 min — Assembly

Editor (CapCut, Descript) or [Editor Pro Max](/boveda/editor-pro-max). You paste everything together.

### 35-40 min — Upload + ManyChat

You post the Reel/TikTok. You activate the ManyChat flow to reply to DMs.

## Anti-patterns

### 1. Wanting to try EVERYTHING before starting

If you wait to "try them all," you end up producing nothing. **Start with Claude + ElevenLabs + ManyChat** (~$57/mo). Add more once you can justify it.

### 2. Skipping the Master Prompt

If you use each tool without orchestration, **you're back to the zoo**. The Master Prompt is what turns this into a **stack**, not a collection.

### 3. Not humanizing the script

The script Claude generates **needs the humanizer**. If you publish raw output, **it sounds like AI**. Run it through [humanize it](/boveda/humanizalo).

### 4. A HeyGen avatar that wasn't trained well

A mediocre avatar = you ruin the video. **Train it with good lighting, several angles, clean audio**.

### 5. ManyChat without a clear flow

If the DM has no specific CTA, **it's noise**. Every DM should lead somewhere (link, lead magnet, calendly).
