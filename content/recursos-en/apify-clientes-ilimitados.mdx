## The problem

Finding customers by hand **burns hours**:
- You open Google Maps
- You copy the name, the phone number
- You hunt for an email on the web
- You jot it down in Excel
- × 50 = a whole day lost

A platform with **1500+ scrapers** (they call them "actors") already built for:

- Google Maps
- LinkedIn
- Instagram
- Facebook
- Amazon

- And almost any site

**You don't code**. You just pick the actor, give it parameters, and get data back.

## Setup in 5 steps

### Step 1 — Free account

`apify.com` → sign up with no card.

**Free plan**: **$5/month in credit**. Enough for hundreds of Google Maps leads if you're just getting started.

### Step 2 — Find your actor

In the Store, search for "Google Maps" (or LinkedIn, Instagram, FB). Pick the one with the most users + best reviews.

### Step 3 — Enable MCP

Avatar (top right) → Account → **Integrations** → enable **MCP Server**.

Apify gives you an authenticated URL.

### Step 4 — Copy the URL

The base URL is `mcp.apify.com` with your token at the end.

### Step 5 — Paste it into Claude

**Claude Desktop**: Settings → Connectors → Add MCP Server → paste the URL.

**Claude Code**:
```bash
claude mcp add apify [url]
```

Restart. In a chat, try:

```
> Can you run Apify's Google Maps actor?
```

If it says yes, you're **connected**.

## The 5 prompts — from zero to a lead list

Paste them **in order**. Wait for a reply before the next one.

### Prompt 1 — Interview to understand your business

```
Before pulling customers with Apify, interview me.

Ask me:
- What I do
- What I sell
- Who my ideal customer is
- What city or country they're in
- How much a new customer is worth to me
- How I contact them today

When you have everything, tell me:
- Which network is best to use (Google Maps, LinkedIn, Instagram, or Facebook)
- Which Apify actor we're going to run

Wait for my OK before running anything.
```

**Without this**, Claude guesses and gives you a generic list. The first prompt **makes the difference**.

### Prompt 2 — Pull 50 leads

```
Using the Apify MCP, pull me a list of 50 potential customers
based on what we agreed on in the interview.

I want a file with:
- Business name
- Contact person if one shows up
- Phone number
- Email
- Website
- Social media
- Address or city
- Any extra useful data

If the actor asks for parameters, show them to me before running.

When it's done, show me the table and save the result to
use in the next steps.
```

### Prompt 3 — Enrich the incomplete ones

```
Take the list of 50 leads and enrich it.

For each contact missing an email, phone, or socials,
try to fill it in by running ANOTHER Apify actor (contact info,
email finder, or a scraper for the social network that business uses).

Clearly mark which data came from the original scraper and which
came from enrichment.

If something stays incomplete, leave it marked as "pending" so we
don't make things up.
```

### Prompt 4 — Personalized emails

```
With the enriched list, draft a personalized first-contact email
for each of the 50.

Use:
- Business name
- Something specific from the scraper (rating, latest post, service they offer)
- Tie it to what I sell

Human tone, short (max 120 words), plain English, with a soft
CTA like "got 10 minutes this week?".

Save the 50 emails in a separate file with a subject and body
per lead.
```

### Prompt 5 — Automate it weekly

```
Set it up to run on autopilot.

Every Monday at 9am:
1. Run the Apify actor again with the same parameters
2. Pull 50 new leads
3. Enrich them
4. Draft the emails

BEFORE sending anything, show me the batch in a message to approve.

If I approve → send via my connected account.
If I DON'T approve → discard.

Let me know how many Apify credits I've spent for the month so I
don't go over the free plan.
```

## Apify vs Scrapling — when to use each

[Scrapling](#) is the **100% free** open-source alternative.

| Aspect | Apify | Scrapling |
|---------|-------|-----------|
| Cost | Free up to $5/month | 100% free, always |
| Setup | Account + MCP URL | Manual local install |
| LinkedIn / Instagram / heavy sites | Handled (IP rotation, captchas) | You get blocked fast |
| Scale | Thousands of leads/month | Dozens to hundreds |
| When | Time is worth more than $5 + complex sites | Starting out + local businesses |

### Rule

- **Starting out + local businesses** → free Scrapling
- **LinkedIn / Instagram / want to scale** → Apify

Both, in that order. Your next customer pays for the account.

## Anti-patterns

### 1. Skipping the interview (Prompt 1)

No interview → generic leads → low conversion → "Apify doesn't work." The problem is the **brief**, not the tool.

### 2. Sending 50 identical emails

Even if you use Apify, if your email is copy-paste, **you'll get flagged as spam**. Personalization IS the point.

### 3. Burning credits on the wrong sites

Pulling 1000 LinkedIn leads when you sell to local businesses = credits wasted. **Pick the right network first**.

### 4. No auth on private sites

Some actors require login. **Store credentials in .env** or in the official integration — never in the prompt.

### 5. Scraping without respecting ToS

Apify handles anti-bot well, but **some sites prohibit commercial scraping**. Check the ToS before operating at large scale.

---

## Compliance and ethics

### What you CAN do

- ✅ Scrape **public** data
- ✅ Contact companies for legitimate B2B
- ✅ Build your own datasets for analysis

### What you CAN'T

- ❌ Scrape **private / login-required** data without authorization
- ❌ Mass unsolicited spam to individuals
- ❌ Violate GDPR (EU) or CCPA (California)
- ❌ Sell personal data without consent

**When in doubt → ask**. Some industries (legal, medical, finance) have strict rules.
