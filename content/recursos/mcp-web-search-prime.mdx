## Qué es

Web Search Prime es la implementación de search de Z.AI como MCP. Provee search web para Claude Code y otros clientes MCP-compatibles usando la infraestructura de Z.AI.

Forma parte del ecosistema Z.AI MCPs ([Vision](/boveda/mcp-zai-mcp-server), [Web Reader](/boveda/mcp-web-reader), [zread](/boveda/mcp-zread)).

## Cómo se instala

```bash
npx claude-code-templates@latest --mcp web/web-search-prime
```

Pre-requisito: cuenta Z.AI + API key.

Para Claude:

```
Instalé el MCP web/web-search-prime. Configurá las credenciales.
Buscame "MCP server" y devolveme los top 10 resultados con title,
URL y snippet.
```

## Qué podés hacer con esto

### Búsqueda web desde Claude

Le pedís a Claude que busque algo, en vez de razonar de memoria. Anti-hallucination para info actual.

### Research recurrente

Searches periódicas sobre temas que cambian. Combinado con scheduler.

### Multi-engine alternative

Si ya usás Z.AI para otros MCPs, mantener stack consolidado.

### Comparado con alternativas

- **[SearXNG](/boveda/mcp-searxng):** privacy-first metasearch, self-hostable
- **[Jina AI](/boveda/mcp-jina-ai):** grounding + fact-checking (más sofisticado)
- **Web Search Prime (este):** Z.AI infrastructure, simpler

## Pre-requisitos

- **Cuenta Z.AI** activa
- **API key**
- **Verificá tools concretas en el JSON**

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **Si ya tenés solution de search funcionando:** evaluá si Z.AI suma algo.
- **Compliance:** servicio chino. Algunos países / orgs tienen restricciones.
- **Para search con citations / grounding:** mejor [Jina AI](/boveda/mcp-jina-ai).
- **Para privacy total:** [SearXNG](/boveda/mcp-searxng) self-hosted.

</Callout>
