## The idea

Claude is a very smart person **locked in a room**. These 4 tools open **4 windows to the world**:

| Superpower | What for |
|------------|----------|
| **Supadata** | Watches and understands videos from any social network |
| **Apify** | Searches and extracts data from all over the internet |
| **Last 30 Days** | Reads the latest news from any niche |
| **Playwright CLI** | Its own browser to research the web |

## Prerequisites

- **Claude Code** installed
- **Node.js 18+**
- **Supadata** free account (supadata.ai)
- **Apify** free account (apify.com)

## Superpower 1 — Supadata (watch videos)

### What it does

An API that **transcribes entire videos** + provides metrics (views, likes, comments). Supports YouTube, TikTok, Instagram, practically any network.

You give it a link → Claude gets the full video content as text, ready to analyze.

### Setup

1. Free account at `supadata.ai`
2. Copy the API key from the dashboard
3. Configure it in `.mcp.json`:

```json
{
  "mcpServers": {
    "supadata": {
      "command": "npx",
      "args": ["-y", "@supadata/mcp"],
      "env": {
        "SUPADATA_API_KEY": "your-key"
      }
    }
  }
}
```

### Use cases

- **Analyze competitors' videos**: what they say, how, what works
- **Structure of viral videos**: break down hook + body + payoff
- **Extract ideas from tutorials**
- **Metrics without opening the platform**

```
> Analyze this viral video: [TikTok URL]

  Tell me:
  1. Structure (hook + body + payoff)
  2. Why it works psychologically
  3. How I could replicate it in my niche [TOPIC]
```

## Superpower 2 — Apify (scrape the internet)

### What it does

1500+ ready-made scrapers (actors) for Google Maps, LinkedIn, Instagram, Facebook, Amazon, etc. Connected to Claude via MCP.

See [Apify unlimited clients](/boveda/apify-clientes-ilimitados) to dive deeper.

### Quick setup

MCP URL: `mcp.apify.com`

Settings → Connectors → Add MCP → paste the URL + your API token.

### Use cases

- **Lead lists** (Google Maps, LinkedIn)
- **Competitor monitoring** (prices, posts, ads)
- **Structured mass research**

```
> Get me 50 [profession] in [city] with phone and email
  using the Apify Google Maps Scraper.
```

## Superpower 4 — Playwright CLI (its own browser)

### What it does

Gives Claude a **real browser** (Chrome). It can:
- Visit URLs
- Click
- Fill in forms
- Take screenshots
- Scroll, hover, etc.

### Installation

```bash
claude mcp add playwright -- npx -y @anthropic-ai/playwright-mcp@latest
```

100% free, no API key.

### Use cases

```
> Open [URL] with Playwright. Log in with [credentials].
  Navigate to /dashboard. Export the "Last 30 days" CSV. Save it
  in /data/.
```

```
> Test the signup flow of my own site http://localhost:3000.
  Take screenshots of each step. Report if anything breaks.
```

See [browse and automate](/boveda/navega-y-automatiza) for Firecrawl + Playwright combined.

## Anti-patterns

### 1. Install 4 and don't use 3

Every tool has a mental maintenance cost. **Install based on real need**.

### 2. Abusing the free Supadata/Apify tiers

The free tier has limits. If you use it a lot, **upgrade** or you'll get throttled.

### 3. Expecting perfect accuracy

Supadata transcribes **very well** but not perfectly. Apify scrapes **very well** but some sites block it. Accept reality.

### 4. Not respecting ToS

Scraping commercial sites can violate terms. **Validate first** before operating at scale.

### 5. No disclosure when using scraped data

If you publish insights based on competitor data, **anonymize** it or ask for permission if you're going to cite them.
