## What it is

This is one of the largest and best-curated marketplaces you'll find for Claude Code. It's maintained by Alireza Rezvani and bundles 192+ skills under 28 themed plugins. The nice part is that it isn't just "engineering": it has separate packs for marketing, product, project management, finance, regulatory compliance (HealthTech/MedTech with ISO 13485, FDA, GDPR), and a C-level pack with a virtual board of directors.

It works in Claude Code and 8 other coding agent environments — if you switch tools tomorrow, the skills come with you.

You don't install all 28 plugins at once (that would be absurd). You add the marketplace and then pick 2-3 packs based on the role you're playing that week.

## How to install it

```bash
# 1) Add the marketplace
/plugin marketplace add alirezarezvani/claude-skills

# 2) Install the packs by role (don't install them all)
/plugin install engineering-skills@alirezarezvani-claude-skills
/plugin install marketing-skills@alirezarezvani-claude-skills
```

Or `/plugin` → Discover.

To have Claude do the curation based on what you need:

```
Install the marketplace alirezarezvani/claude-skills. I'm a [your role: full-stack
dev / founder / PM / etc.] working on [your project]. Which 2-3 of the 28 available
packs should I install first, and why?
```

## What you can do with it

### Cover a new role in a week

If this week you have to act as a Project Manager (your PM left), you install `pm-skills` and suddenly Claude knows how to write Jira tickets following the pack's convention, put together a sprint plan, and facilitate a retro. It doesn't replace a senior PM, but it fills the gap.

### Ship content without a marketing team

`marketing-skills` comes with 43 skills grouped into 7 pods. For a solo founder: asking Claude to use the Content + SEO + CRO pack gets you an editorial calendar, an optimized landing page, and a conversion analysis in the same session.

### Heavy compliance (HealthTech)

If you work on something regulated (`ra-qm-skills`), the 13 regulatory skills cover ISO 13485, EU MDR, FDA 510(k), CAPA management. It won't validate you with the FDA, but it saves you months of research on what each framework requires.

### Stress-testing a plan with `executive-mentor`

Like [Adversarial Spec](/boveda/spec-kit), but for business decisions: you present your plan and the skill critiques it with hard questions. Useful before a board meeting or an irreversible decision.

### A virtual C-level board

`c-level-skills` activates synthetic "advisors" (CEO, CTO, COO, CPO, CMO, CFO, CRO, CISO, CHRO) who give you opinions from their own angles. They don't replace real humans, but they help structure your thinking.

## Real benefits

- **Genuine horizontal coverage** — a single marketplace covers eng/marketing/product/finance. Few do that.
- **Cross-tool** — the day you try Cursor or Codex, you don't lose the skills.
- **Active maintenance** — 7.5k stars aren't an accident. The author keeps iterating.
- **Granular** — you install only the packs for the role you're playing this week.

## When it's NOT a good fit

<Callout type="warning" title="Before installing">

- **Don't install all 28 packs at once:** you'll have 192 skills loaded and Claude will get dizzy. Pick 2-3 by role.
- **If you already pay for dedicated tools** (Notion AI for marketing, Linear AI for PM), the skills overlap. Try one before duplicating.
- **`ra-qm-skills` doesn't certify you for anything:** they're guides and templates. Regulatory validation is still a human job.
- **`c-level-skills` doesn't replace real mentors:** use it to structure ideas, not to decide on your own.

</Callout>
