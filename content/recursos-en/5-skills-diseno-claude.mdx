## Skill 1 — Frontend Design

### What it does

You tell Claude **what your page/app is for** and it designs everything:

- Complete layout
- Color palette
- Typography
- Animations and micro-interactions
- Real, ready-to-use code (not a mockup or an image)

400k+ devs already use it. It's the **most popular** official skill in the catalog.

### Installation

```bash
claude install-skill https://github.com/anthropics/claude-code/tree/main/plugins/frontend-design/skills/frontend-design
```

### Examples

**Landing page for a bakery**:

```
> Design a landing page for an artisan bakery. Warm colors,
  elegant typography, smooth animations on scroll.
  Include a hero with a photo, menu, testimonials, contact form.
```

**Minimalist dashboard**:

```
> Design a minimalist dashboard for a personal finance app.
  Neutral colors, clean typography, cards with soft shadows.
  Include a spending chart, transaction list, monthly summary.
```

### When it shines

- You need a page/app with **cohesive aesthetics**
- You're not a designer and you want something better than the generic default
- You're fine with code-first (it hands you React/HTML, not Figma)

### When NOT

- You need visual iteration with a client (collaborative Figma is better)
- Your brand requires a very specific identity (load [Brand Guidelines](/boveda/cinco-habilidades-marketing) first)

## Skill 2 — Canvas Design

### What it does

Generates posters, covers and graphics for social. Output: PNG or PDF ready to download.

Unlike Canva (templates that look like everyone else's), Canvas Design generates a **unique design** that respects the brief.

### Installation

```bash
claude install-skill https://github.com/anthropics/skills/tree/main/skills/canvas-design
```

### Examples

**Instagram poster for a course**:

```
> Poster for Instagram announcing my online photography course.
  Dark colors with gold accents. Just the course name and date.
  Minimal text, maximum visual impact.
```

**YouTube thumbnail**:

```
> YouTube thumbnail for a tech channel. Modern, clean style,
  dark blue with cyan details. Channel name and a short tagline.
```

### When it shines

- You produce visual content regularly (Reels, Posts, Stories)
- You need something more professional than generic Canva
- You want to iterate on variants fast

### When NOT

- For video and motion graphics — you need another tool ([Editor Pro Max](/boveda/cinco-habilidades-marketing) or [Higgsfield MCP](/boveda/higgsfield-mcp))
- Enterprise production where you need granular traceability for each asset

## Skill 3 — Web Artifacts Builder

### What it does

Builds **small functional apps** without code. 40+ pre-built components, React + TypeScript + Tailwind stack. Output: an executable HTML file.

### Installation

```bash
claude install-skill https://github.com/anthropics/skills/tree/main/skills/web-artifacts-builder
```

### Examples

**Pricing calculator**:

```
> Pricing calculator for a graphic design service.
  Inputs: project type (logo, poster, web), complexity, urgency.
  Show the estimated price instantly.
```

**Task organizer**:

```
> Task organizer with 3 columns: To Do, In Progress,
  Done. Drag and drop between columns. Minimalist design.
```

### When it shines

- You need a small **utility tool** (calculator, form, generator)
- A quick prototype that **works** (not just a mockup)
- A tool for internal team use without wanting to deploy a backend

### When NOT

- An app that requires a real backend (DB, auth, APIs)
- An app going to production with many users — you need a more solid stack
- Complex business logic (>500 LOC)

### Examples

**Animated natural fractal**:

```
> Generative art inspired by natural growth patterns —
  tree branches, roots, coral. Greens and golds over
  a dark background. Controls to adjust density and speed.
```

**Particle system**:

```
> Particle system with geometric patterns flowing like a river.
  Cyan and magenta over black. Controls for the number of particles
  and the strength of the flow.
```

### When it shines

- Animated backgrounds for premium landing pages

- Traditional branding (standard vector logos are better)

## The combined flow

You need to launch a landing page for an online course.

```
1. Theme Factory:
   > Generate a theme for a professional photography course landing.
     Audience: serious hobbyists. Mood: aspirational + premium.

2. Frontend Design (with the theme active):
   > Design the complete landing: hero, problem, solution, course
     modules, testimonials, FAQ, final CTA. Apply the theme.

3. Canvas Design:
   > Generate 5 Instagram posters to promote the course, using
     the same theme.

4. Algorithmic Art:
   > Generate a subtle animated background for the hero, aligned with
     the mood (not loud, decorative).

5. Web Artifacts Builder:
   > Interactive calculator: "what level are you" — 5 questions →
     recommends a starting module. Embeddable in the landing.
```

**Total output**: landing + posters + animation + interactive tool. **All visually coherent** because you started with Theme Factory.

Estimated time: 2-3h. Without the 5 skills: 1-2 days + a designer.

## Anti-patterns

### 1. Activating all 5 at the start of the project

Each skill takes up context. Activate **one at a time** as you need it. After launch, keep active only the ones you use regularly.

### 2. Algorithmic Art for functional UIs

The temptation: "it looks nice, I'll put it in the dashboard". **Mistake**. Generative art in a functional UI **distracts the user**. Save it for the hero, decorative backgrounds, splash screens.

### 3. Frontend Design without a defined theme

If you ask "design me a landing" without a theme loaded, Claude uses defaults (blue + Inter). It'll come out correct but generic. **Load Theme Factory first**.

### 4. Web Artifacts for real apps

Web Artifacts Builder is **excellent for small tools**. For an app going to production with real users and persistent data, use a full stack (Next.js + DB + Auth).

### 5. Canvas Design instead of real photos

For a physical product, **a real photo > abstract design**. Canvas Design is for promos/announcements, not for replacing product photography.
