# Vincent Conace — Portfolio

Sitio personal de Vincent Conace, emprendedor desde Mar del Plata, Argentina.
Soluciones de software a medida y automatizaciones con IA para negocios y emprendedores.

## Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [GSAP](https://gsap.com/) + [Lenis](https://github.com/studio-freight/lenis) (smooth scroll)

## Desarrollo

```bash
pnpm install
pnpm dev
```

El sitio queda disponible en `http://localhost:3000`.

## Variables de entorno

Crear un archivo `.env.local` con:

```
NEXT_PUBLIC_WHATSAPP_NUMBER=54XXXXXXXXXX
```

Esto se usa para construir el link de WhatsApp del formulario de contacto.
