## Qué es

Es el marketplace oficial que Anthropic mantiene específicamente para [Claude Cowork](/boveda/dominar-cowork) — no para Claude Code de programación, sino para el "compañero de trabajo" de oficina. Si tu mundo es Gmail + Calendar + Notion + Slack + Drive y querés que Claude se mueva fluido en esas herramientas, este es **el** marketplace.

39 plugins en total, organizados por rol/función. La filosofía es verticalizar: en vez de "Claude que hace de todo", te da un pack específico por rol (sales, finance, legal, marketing, etc.) que ya viene con los flujos y las plantillas para ese trabajo.

11k stars indica que la comunidad lo adoptó masivamente. Y como es de Anthropic, tiene la garantía oficial.

## Cómo se instala

```bash
# 1) Marketplace
/plugin marketplace add anthropics/knowledge-work-plugins

# 2) Instalá los packs según tu rol
/plugin install productivity@anthropics-knowledge-work-plugins
/plugin install enterprise-search@anthropics-knowledge-work-plugins
/plugin install sales@anthropics-knowledge-work-plugins  # o marketing, finance, etc.
```

O `/plugin` → Discover.

Para Claude (en Cowork):

```
Instalá el marketplace anthropics/knowledge-work-plugins. Mi rol es
[ventas / marketing / finanzas / etc.] y mi stack es [Gmail + Notion +
Slack + HubSpot, por ejemplo]. Recomendame los 3-4 packs para empezar.
```

## Qué podés hacer con esto

### Tener un Cowork productivo desde el día 1

El pack `productivity` arma tasks + calendar + memoria de contexto. El pack `enterprise-search` te deja buscar en todo a la vez. Con esos dos solos ya pasás de "Claude responde preguntas" a "Claude trabaja con vos en la oficina".

### Cubrir un rol entero sin contratar especialista

Si tu equipo no tiene PM, instalás `product-management` y Claude te asiste con specs, roadmaps, síntesis de user research. Si no tenés legal, `legal` te ayuda con NDAs y contract review. No reemplaza al especialista, pero te tapa muchos huecos.

### Brand voice consistente automáticamente

`brand-voice` (de Tribe AI) descubre tu voz desde docs existentes, genera guidelines y valida contenido generado por IA. Útil para equipos de marketing que se cansaron de revisar manualmente cada salida.

### PDFs con anotaciones reales

`pdf-viewer` te deja ver y anotar PDFs adentro de Cowork. Mark up de contratos, firmas, formularios. Sin tener que abrir otra app.

### Integrar con tu stack sin escribir MCPs

Salesforce Slack, Apollo, ZoomInfo, Mintlify, Zapier — todos verificados por Anthropic. Conectás sin pelearte con setup ni preocuparte por seguridad.

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **Si usás Claude Code (no Cowork):** los packs están pensados para Cowork. Mucho de lo que ofrecen no aplica a un dev en terminal.
- **Si no estás en plan Claude Max:** [Cowork requiere plan Max](/boveda/dominar-cowork). Verificá antes.
- **No instales los 39 packs juntos:** vas a tener 118 skills cargadas. Elegí 3-4 según rol.
- **Solapamiento con [Claude Small Business](/boveda/claude-small-business):** Small Business también cubre roles. Mirá cuál te encaja antes de instalar ambos.
- **Algunas integraciones requieren cuentas pagas** (Apollo, ZoomInfo, Mintlify). El plugin conecta — la cuenta la pagás vos.

</Callout>
