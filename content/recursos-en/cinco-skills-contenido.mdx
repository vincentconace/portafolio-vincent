## The premise

You used to need **5 people** to make content:
- A researcher for topics
- A designer for images
- A video editor
- A web designer
- A writer who doesn't sound like a robot

These 5 skills let you do **the work of all 5**. Without paying anything extra.

| 01 | Notebook LM | Researcher |
| 02 | Banana Cloud | Image designer |
| 03 | Editor Pro Max | Video editor |

## 01 — Notebook LM (the researcher)

### What it does

A magic box: you throw articles, videos, PDFs — whatever — at it and it spits out a **structured summary** with the key points.

No more spending hours reading 50 sources to find your angle.

### Flow

```
1. You give it your sources (links, PDFs, videos)
2. It processes them and hands you a summary with key points
3. You use that summary as the base for your next video/post
```

### Installation

```bash
pip install notebooklm-py
```

Repo: `teng-lin/notebooklm-py`.

### What it's for

When you're going to make content on a topic you don't fully know. You give it 5-10 reliable sources and it builds the "what matters" before you even start writing.

## 02 — Banana Cloud (the designer)

### What it does

You **describe** the image you need — a product photo, a graphic for social media, an infographic — and it generates it in seconds.

No Canva, no Photoshop. Straight from Claude.

### Installation

```bash
git clone https://github.com/AgriciDaniel/banana-claude.git && cd banana-claude
```

### Use cases

```
"Minimalist product photo for my sneaker e-commerce store"
"Instagram banner with a gradient background and bold text"
"Infographic comparing 3 features"
```

### Difference vs Claude Banana

Banana Cloud is the **AgriciDaniel** version focused on the pipeline. There's also [Claude Banana](/boveda/claude-banana) (by Hainrixz) focused on the narrative format of prompts. Both are valid.

## 05 — Humanize It (the authentic voice)

### What it does

You paste in any AI text and it hands it back sounding **like you would write it**. Zero robotic phrases, zero weird em dashes, zero "in today's landscape...".

### Why it closes the loop

The other 4 skills generate content for you. But if everything comes out with an AI tone, **it shows**. Humanize It is the final filter.

## Anti-patterns

### 1. Skipping Humanize It

If you publish without humanizing, **it shows**. It's the step people skip and the one that matters most.

### 2. Notebook LM with garbage sources

Garbage in, garbage out. **Curate the sources** before feeding them in — Notebook LM summarizes, it doesn't judge quality.

### 3. Banana without a defined style

"A nice image" → generic output. **Specify the style, palette, mood**.

### 4. Editor Pro Max for long video

It's meant for short pieces (reels, shorts, explainers). For long video (10+ min), use a real editor.

### 5. Stitch for direct production

Stitch designs for you, it doesn't build production-ready. **Use it as a guide**, not as the final deliverable.
