## The combination

Replicating the design of any web page requires 2 pieces:

| Piece | Role |
|-------|-----|
| **UI UX Pro Max Skill** | Knowledge — 161 design rules by industry |
| **The replication prompt** | Instruction — analyze the URL and replicate everything |

Result: you paste a URL → Claude analyzes layout/colors/fonts/animations → rebuilds it in your project with **your content**.

---

## Prerequisites

- **Claude Code** installed ([install Claude Code](/boveda/instalar-claude-code))

## Install UI UX Pro Max

```bash
claude install-skill https://github.com/nextlevelbuilder/ui-ux-pro-max-skill
```

### What it includes

| Component | Amount |
|------------|----------|
| UI styles | 67 (Glassmorphism, Neumorphism, Brutalism, AI-Native, and more) |
| Color palettes | 161 (by product category) |
| Font combos | 57 (with Google Fonts) |
| Reasoning rules | 161 across 8 domains (tech, finance, health, e-commerce, and more) |

## The replication prompt

```
This is the landing page [URL].

Replicate:
- Layout and spacing
- Typography
- Animation patterns
- Colors and palette

Use the UI UX Pro Max skill and Tailwind CSS to build a
fully responsive site.

The templates should be applicable to other pages/landings,
so that my landing looks like this site but with my
content and copy.

If you don't have images or graphics, use NanoBanana to generate them.
```

## Anti-patterns

### 1. Replicating without legal permission

Even though the code you generate is 100% yours, **cloning protected designs from big brands** is a gray area. For personal use/experimentation, OK. For a commercial product, **draw inspiration, don't copy literally**.

### 2. Expecting pixel-perfect

The skill replicates **patterns and aesthetics**, not an exact copy. If you need pixel-perfect, use the [Cloner](/boveda/clonador-de-paginas) or do it by hand.

### 3. Not using your own brand

Replicating and leaving the **original's colors and fonts** = your brand looks generic. Change it afterwards to your palette.

### 4. Replicating without understanding what you're replicating

The skill generates code. **Read it**. If you don't understand something, ask Claude to explain. It's **learning**, not just copy-paste.

### 5. Ignoring the original's performance

Sometimes the original site is **pretty but heavy**. If you replicate everything including the weight, **you'll have the same problem**. Optimize whatever applies.
