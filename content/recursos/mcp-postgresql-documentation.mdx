## Qué es

Este MCP no conecta a tu DB. Conecta a la **documentación oficial de PostgreSQL**. La sacada: cuando Claude va a escribir SQL o hablar de features Postgres, consulta los docs reales en vez de tirar de memoria.

Es similar a [Context7](/boveda/mcp-context7) (que cubre librerías genéricas) pero especializado en Postgres. Y complementa a [pg-aiguide](/boveda/plugin-pg-aiguide) — pg-aiguide es un plugin oficial de Timescale con skills y MCP; este MCP es más focal: solo docs.

Útil cuando:
- Trabajás con features menos comunes (window functions, recursive CTEs, JSON operators avanzados)
- Estás migrando entre versiones y querés saber qué cambió
- Querés que Claude no invente sintaxis o flags

```bash
npx claude-code-templates@latest --mcp database/postgresql-documentation
```

Para Claude:

```
Instalé el MCP database/postgresql-documentation. Quiero usar
LATERAL JOIN en una query compleja. Consultame los docs y mostrame
la sintaxis correcta + 2 ejemplos representativos.
```

## Qué podés hacer con esto

### Anti-hallucination en SQL avanzado

"Para implementar paginación eficiente sobre 100M rows, ¿qué patron usa Postgres? Consulta los docs y mostrame el approach correcto con keyset pagination". Claude cita docs en vez de inventar.

### Aprender features nuevas

"En Postgres 16, ¿qué hay nuevo en JSON operations vs 14? Listame las features con ejemplos".

### Resolver dudas sintácticas precisas

"¿Cuál es la sintaxis correcta de `GENERATED COLUMNS` en Postgres? Identificá la versión mínima requerida".

### Migration helpers

"Estoy migrando una query de MySQL a Postgres. La query usa `GROUP_CONCAT`. ¿Cuál es el equivalente en Postgres y cuál es la nota de performance?".

### Optimización con index types

"¿Cuándo conviene un GiST index vs GIN vs BTREE? Consultame los docs y dame árbol de decisión".

## Pre-requisitos

- **Node.js + Claude Code** funcionando
- **Conexión a internet** (consulta docs online)

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **Si solo necesitás queries básicas:** Claude conoce SQL básico de memoria. El MCP suma valor en features avanzadas.
- **Si usás [pg-aiguide](/boveda/plugin-pg-aiguide):** ya incluye docs + skills + MCP. Verificá overlap.
- **Si tu DB no es Postgres:** no aplica.
- **Para data operations en tu DB:** este MCP es solo docs. Combinalo con [postgresql-integration](/boveda/mcp-postgresql-integration).

</Callout>
