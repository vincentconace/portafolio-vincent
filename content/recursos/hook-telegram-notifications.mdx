## Qué es

La versión minimal del pipeline Telegram. Cuando Claude o un subagent terminan, manda mensaje sin detalle.

Hook events: `Stop` + `SubagentStop`.

Pre-requisitos:
- `TELEGRAM_BOT_TOKEN` (via @BotFather)
- `TELEGRAM_CHAT_ID`

## Cómo se instala

```bash
npx claude-code-templates@latest --hook automation/telegram-notifications
```

Configurá:

```bash
export TELEGRAM_BOT_TOKEN="..."
export TELEGRAM_CHAT_ID="..."
```

## Qué podés hacer con esto

### Aviso al celular

Telegram tiene la mejor mobile UX. Notification al cel cuando termina.

### Personal channel

Bot privado con vos mismo = canal de tracking sin team.

### Subagent completions

Si tu workflow lanza subagents, cada uno te avisa.
