## The uncomfortable truth

If you don't know how to talk to Claude, **not even the $200 plan will work for you**.

The secret isn't the plan — it's learning to:
- Structure your ideas before writing
- Be specific, not conversational
- Switch models depending on the task
- Apply the [5 common mistakes](/boveda/cinco-errores-claude)

With those habits, **the $20 Pro plan performs incredibly well**. Without them, even the $200 one runs out on you.

---

## The 4 plans

| Plan | Price | What it's for |
|------|--------|----------|
| **Free** | $0 | Trying Claude (not for real use) |
| **Pro** | $20/mo | Getting started or casual daily use |
| **Max 5x** | $100/mo | Working with Claude all day |
| **Max 20x** | $200/mo | You never stop (literally) |

---

## Free plan

### What's included

- Basic web and mobile access
- ~5-20 messages/day (depends on length)
- **Does NOT include Claude Code**
- No advanced features

### When it's worth it

- **Just to try** Claude
- See if you like how it thinks
- Decide whether you're going to pay

### When NOT

- For productive use (you'll hit limits within 1h)
- If you plan to integrate it into a real workflow

**Verdict**: It's for getting to know it, not for using it. If you like it, move up to Pro.

---

## Pro plan — $20/mo

### What's included

- **Claude Code** (the agent that codes)
- **~45 messages every 5h**
- Unlimited projects
- Models: **Sonnet + Haiku**
- Web search and research
- Basic connectors

### The typical mistake

> "Pro isn't enough, I'm moving up to Max"

90% of the time the problem **isn't the plan**. It's that you're making the [5 common mistakes](/boveda/cinco-errores-claude):

1. Correction messages (instead of editing)
2. Long chats with no reset
3. Separate questions
4. Active connectors you're not using
5. Working during peak hours

**Before upgrading, fix those habits**. You'll double how long the plan lasts without changing anything.

### When it's worth it

- You're starting out with Claude
- Casual daily use (1-3h)
- Mid-sized Claude Code tasks
- Learning how to talk to it

### When to move up to Max

Only if:
- You apply the 5 fixes and STILL run out of quota
- You work >5h/day with Claude Code seriously
- You regularly need Opus for complex tasks

**Verdict**: Stay here until you **master** how to talk to it. Then reassess.

## Max 5x plan — $100/mo

### What's included

- **5x more usage** than Pro
- Access to **Opus** (the most capable model)
- **Computer Use** (controlling your computer)
- Automations and long tasks
- Priority on new features
- **All integrations** (Gmail, Drive, Notion, etc.)

### What changes vs Pro

**Opus** is the biggest change. For complex reasoning:
- Architecture design
- Debugging race conditions
- Product decisions with deep analysis

Opus is **objectively better**. Pro doesn't have it.

**Computer Use** lets Claude operate your computer while you do something else.

### When it's worth it

- You work full-time with Claude Code (>30h/week)
- You use Computer Use for automations
- Your work requires complex reasoning (not just generating code)
- Your company pays for you (the investment vs salary is easy to justify)

### When NOT

- If Pro is enough with the 5 fixes applied
- If you just want "more messages" without a clear reason for Opus

**Verdict**: For serious work. Claude becomes an all-day teammate.

---

## Max 20x plan — $200/mo

### What's included

- **20x more usage** than Pro (practically unlimited)
- Everything in Max 5x without restrictions
- Priority access to experimental features
- Long sessions (hours) without hitting the quota

### When it's worth it

Honestly: **very few people**.

- You don't sleep and use Claude 12-16h/day
- You're part of a technical team that delegates heavily to Claude
- You sell services with Claude (AI consultancy, technical agency)
- You want to have "everything" without thinking about limits

### When NOT

If on Max $100 you **don't hit the limit every day**, you don't need $200.

**Verdict**: Only if you don't sleep. If you don't reach the $100 limit, $200 is a waste.

## Mistakes when choosing a plan

### 1. "I'll go straight to Max so I don't have to worry"

If you haven't learned to talk to Claude, $100 goes just as fast. **Start on Pro, learn, scale up if you need to**.

### 2. "Free is enough because I use it a little"

Free **doesn't have Claude Code**. If you want to use the tool seriously, Pro is the minimum.

### 3. "Max $200 is 2x Max $100, so it's better"

False. Max $200 is **20x Pro** vs **5x Pro for Max $100**. It's 4x more quota, not 2x. But **if you don't use the 5x**, the 20x is completely overkill.

### 4. "I'll go with the annual plan to save money"

If you're still exploring whether Claude is for you, **go monthly**. Annual saves ~15% but locks you in for 12 months.

### 5. "I'll switch between plans depending on the month"

Possible but **operationally annoying**. Better to find the plan that matches your average usage and stick with it.

## When to scale up (clear signals)

**From Free to Pro**:
- Daily limits are holding you back
- You want to try Claude Code

**From Pro to Max $100**:
- You applied the 5 fixes and still max out your quota
- You need Opus regularly
- You want Computer Use

**From Max $100 to Max $200**:
- You hit the limit **daily**
- You do 8+ hour sessions with Claude
- The extra $100/mo would pay for itself with 1h of your time saved

If the signals aren't clear, **don't upgrade**.
