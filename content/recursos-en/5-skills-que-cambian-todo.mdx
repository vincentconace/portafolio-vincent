## The analogy

Using Claude without skills is like a **phone without apps**. It technically works — but you're missing 90% of the potential.

These 5 are the "essential apps." Free. They install in seconds. They work for beginners and experts alike.

| # | Skill | Function |
|---|-------|---------|
| 01 | Skill Creator | Your personal App Store — creates skills |
| 02 | Humanize It | Kills the AI tone in any text |
| 03 | Marketing Skills | 40+ marketing tools in one |
| 04 | UI UX Pro Max | Design with 67 pro styles |
| 05 | Superpowers | Autopilot with subagents |

## 01 — Skill Creator (the factory)

### What it is

Claude's App Store. An **official Anthropic** skill for creating, testing, and evaluating your own skills.

You describe what you want in plain language, and in under 5 minutes you have a skill ready.

### Why it comes first

With this one you can **create the others**. If a skill doesn't exist, you build it yourself.

### Installation

```bash
claude install-skill https://github.com/anthropics/skills/tree/main/skills/skill-creator
```

### Example

```
/skill-creator I want a skill that generates professional PDF reports of my meetings. It takes notes in text and builds a PDF with sections, key points, and next steps.
```

See [skill creator](/boveda/creador-de-habilidades) to dig deeper.

## 02 — Humanize It (kills the AI tone)

### What it is

It detects more than 40 AI writing patterns and **removes** them. It evaluates across 6 dimensions (vocabulary, structure, formatting, tone, communication, signals) and rewrites.

### The insight

"In today's landscape...", "It's crucial to highlight...", "Without a doubt..." — every AI text has the same turns of phrase. Humanize It hunts them down and kills them.

### Installation

```bash
git clone https://github.com/Hainrixz/humanizalo.git ~/.claude/skills/humanizalo
```

### Example

```
/humanizalo In today's digital marketing landscape, it's crucial to highlight that content strategies play a vital role...
```

See [Humanize It](/boveda/humanizalo) for the details.

## Anti-patterns

### 1. Installing all 5 on day 1

You pile on context that Claude reads in every response. **Install one, master it, then the next.**

### 2. Humanizing EVERYTHING

For technical docs or contracts, humanizing can **break precision**. Save it for marketing/blog/social.

### 3. Marketing Skills without context

40+ skills is a lot. If you ask for generic "marketing," the output is generic. **Be specific** about the product, audience, and goal.

### 4. UI UX Pro Max without picking a style

If you don't tell it which style you want, Claude picks any of them. **Ask for a specific style** (brutalist, Notion-like, glass).

### 5. Superpowers for small tasks

For a 20-line script, Superpowers is overkill. Save it for **real projects** with several moving pieces.
