## Qué es

Nia MCP vive en la categoría `deepresearch` del catálogo aitmpl — junto con tools enfocadas a investigación profunda con AI. La descripción inline es escueta — para detalles concretos de qué expone, el [JSON de configuración](https://github.com/davila7/claude-code-templates/blob/main/cli-tool/components/mcps/deepresearch/mcp-server-nia.json) es la fuente más confiable.

128 installs lo posicionan como uno de los MCPs adoptados de su categoría nicho.

```

```

## Qué podés hacer con esto

### Research depth

La categoría `deepresearch` se ocupa de investigación profunda con AI: workflows multi-step, agents que indagan en múltiples fuentes, síntesis de hallazgos.

Si tu uso es research académico, due diligence o análisis profundo de tema, este MCP encaja en la categoría.

### Validar el alcance real

Para entender exactamente qué hace Nia, lo más útil:
1. Instalá el MCP
2. Pedile a Claude que liste las tools que expone
3. Probá una llamada simple para ver el output

## Pre-requisitos

- **Verificá el JSON de configuración** para entender qué exige (credenciales, env vars)

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **Información limitada en aitmpl:** la descripción es genérica. Si tu uso es serio, mirá la doc oficial del proveedor.
- **128 installs es medio:** comunidad probando moderada.
- **Para research académico específico** ([arXiv MCP](/boveda/mcp-arxiv-mcp-server) para papers), [Jina AI](/boveda/mcp-jina-ai) para grounding) puede ser más concreto.
- **Probá en sandbox** antes de uso productivo.

</Callout>
