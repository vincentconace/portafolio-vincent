## Qué es

Footballbin Predictions es un servicio que provee pronósticos AI sobre partidos de fútbol — específicamente Premier League inglesa y Champions League. El MCP enchufa ese servicio a Claude Code. Le pedís predicciones, te devuelve:

- Marcador half-time
- Marcador full-time
- Próximo goleador
- Cantidad de corners

Es nicho. Si construís apps de fantasy football, content sobre fútbol, o agents que tracken matches, te ahorra escribir el cliente de la API.

## Cómo se instala

```bash
npx claude-code-templates@latest --mcp integration/footballbin-predictions
```

Pre-requisito: cuenta en Footballbin y API key (si aplica).

Para Claude:

```
Instalé el MCP integration/footballbin-predictions. Pediré la predicción
del próximo partido del Real Madrid en Champions: marcador half-time,
full-time, próximo goleador y corners.
```

## Qué podés hacer con esto

### Content automatizado

"Generame un post para Instagram con la predicción del partido X de hoy: visual del marcador esperado, top scorer estimado, corner count". Pipeline content para cuentas de fútbol.

### Apps fantasy

Si tu app de fantasy necesita scores estimados o probabilidades, el MCP da el feed. Útil para sugerir a usuarios qué jugadores levantar.

### Análisis comparativo

"Comparame las predicciones del MCP vs los resultados reales de los últimos 10 partidos del Liverpool. ¿Cuál es la accuracy?". Útil para validar antes de usar en producción.

### Bots / agents temáticos

Agent que tuitea predicciones antes de cada partido, las contrasta con el resultado, y arma un score de accuracy semanal.

## Pre-requisitos

- **Cuenta Footballbin** activa
- **API key** si aplica
- **Foco en Premier League / Champions:** otras ligas no están cubiertas

## Cuándo NO conviene

<Callout type="danger" title="Antes de usar para apostar dinero">

- **NO son recomendaciones de apuesta.** Son predicciones AI. Pueden estar mal. Usalas para entretenimiento, análisis o contenido — no para tirarle plata sin criterio propio.
- **Cobertura limitada:** solo Premier League + Champions. Si tu interés es La Liga, Bundesliga, Serie A, NBA, NFL, etc., este MCP no aplica.
- **Accuracy desconocida:** no hay benchmark público. Validá con histórico antes de confiar.
- **Apuestas y legalidad:** apostar es regulado por país. Vos sos responsable.
- **Solo 51 installs:** poca comunidad. Probá antes en sandbox.

</Callout>
