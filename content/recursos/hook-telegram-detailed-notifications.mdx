## Qué es

Equivalente a [Discord Detailed](/boveda/hook-discord-detailed-notifications) y [Slack Detailed](/boveda/hook-slack-detailed-notifications) para Telegram. Manda mensajes con info detallada:
- Working directory
- Duración
- System info

Hook events: `SessionStart` + `Stop`.

Pre-requisito: bot de Telegram + chat ID donde mandar (creás bot via @BotFather).

```bash

```

Pre-requisitos:
- Bot creado con @BotFather (recibís token)
- Chat ID (canal o personal)

```
Instalé el hook automation/telegram-detailed-notifications. Configurá

```

## Qué podés hacer con esto

### Notifications al celular

Telegram tiene la mejor experiencia mobile de las tres plataformas. Para tracking on the go.

### Equipos en países donde Telegram domina

LATAM, Europa del Este, partes de Asia. Si tu equipo está ahí, Telegram > Slack.

### Personal channels

Bot privado con vos = tu propio canal de tracking sin team noise.

### Combo con [Telegram Error Notifications](/boveda/hook-telegram-error-notifications)

Detailed para success + Error para problems = visibility completa.

## Cuándo NO conviene

<Callout type="warning" title="Antes de instalar">

- **Si no usás Telegram:** Slack o Discord.
- **Bot setup overhead:** crear bot, conseguir chat ID, configurar. Si es uso casual, no vale.
- **Compliance:** Telegram tiene policies. Para teams enterprise, verifiquen.
- **Spam:** sesiones cortas saturan tu chat.

</Callout>
