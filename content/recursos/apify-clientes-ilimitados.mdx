## El problema

Buscar clientes a mano **quema horas**:
- Entrás a Google Maps
- Copiás nombre, teléfono
- Buscás email en la web
- Anotás en Excel
- × 50 = todo el día perdido

---

## Qué es Apify

Plataforma con **1500+ scrapers** (le dicen "actors") ya armados para:

- Google Maps
- LinkedIn
- Instagram
- Facebook
- Amazon
- YouTube
- TikTok
- Y casi cualquier sitio

**Vos no programás**. Solo elegís el actor, das parámetros, recibís data.

**Lo nuevo**: Apify tiene **MCP oficial**. Lo conectás a Claude una vez, le hablás en español: *"sacáme 50 dentistas de CDMX con teléfonos y correos"* → Claude corre el actor correcto, limpia info, te entrega lista.

## Setup en 5 pasos

### Paso 1 — Cuenta gratis

`apify.com` → registro sin tarjeta.

**Plan gratis**: **$5/mes de crédito**. Suficiente para cientos de leads de Google Maps si recién arrancás.

### Paso 2 — Buscar tu actor

En el Store buscás "Google Maps" (o LinkedIn, Instagram, FB). Elegí el que tenga más usuarios + mejores reseñas.

### Paso 3 — Activar MCP

Avatar (arriba derecha) → Account → **Integrations** → activá **MCP Server**.

Apify te da URL autenticada.

### Paso 4 — Copiar URL

La URL base es `mcp.apify.com` con tu token al final.

### Paso 5 — Pegar en Claude

**Claude Desktop**: Settings → Connectors → Add MCP Server → pegás URL.

**Claude Code**:
```bash
claude mcp add apify [url]
```

Reiniciá. En un chat probá:

```
> ¿Podés correr el actor de Google Maps de Apify?
```

Si dice que sí, **conectado**.

## Los 5 prompts — de cero a lista de leads

Pegalos **en orden**. Esperá respuesta antes del siguiente.

### Prompt 1 — Entrevista para entender tu negocio

```
Antes de sacar clientes con Apify, entrevistame.

Preguntame:
- A qué me dedico
- Qué vendo
- Quién es mi cliente ideal
- En qué ciudad o país están
- Cuánto vale para mí un cliente nuevo
- Cómo los contacto hoy

Cuando tengas todo, decime:
- Qué red conviene usar (Google Maps, LinkedIn, Instagram o Facebook)
- Qué actor de Apify vamos a correr

Esperá mi OK antes de ejecutar nada.
```

**Sin esto**, Claude adivina y te da lista genérica. El primer prompt **marca la diferencia**.

### Prompt 2 — Sacar 50 leads

```
Usando el MCP de Apify, sacame lista de 50 clientes potenciales
según lo que acordamos en la entrevista.

Quiero archivo con:
- Nombre del negocio
- Persona de contacto si aparece
- Teléfono
- Correo
- Sitio web
- Redes sociales
- Dirección o ciudad
- Cualquier dato extra útil

Si el actor pide parámetros, mostrámelos antes de correr.

Cuando termine, mostrame la tabla y guardá el resultado para
usarlo en los siguientes pasos.
```

### Prompt 3 — Enriquecer los incompletos

```
Tomá la lista de 50 leads y enriquecela.

Para cada contacto que le falte correo, teléfono o redes,
intentá completar corriendo OTRO actor de Apify (contact info,
email finder, o scraper de la red social que use ese negocio).

Marcá claramente qué dato es del scraper original y cuál salió
del enriquecimiento.

Si algo queda sin completar, dejalo marcado como "pendiente" para
no inventar.
```

### Prompt 4 — Correos personalizados

```
Con la lista enriquecida, redactá correo de primer contacto
personalizado para cada uno de los 50.

Usá:
- Nombre del negocio
- Algo específico del scraper (rating, último post, servicio que ofrecen)
- Conectá con lo que yo vendo

Tono humano, corto (max 120 palabras), español neutro, con CTA
suave tipo "¿tenés 10 min esta semana?".

Guardá los 50 correos en archivo separado con asunto y cuerpo
por lead.
```

### Prompt 5 — Automatizar semanal

```
Dejalo corriendo en automático.

Cada lunes 9am:
1. Corré de nuevo el actor de Apify con mismos parámetros
2. Sacá 50 leads nuevos
3. Enriquecelos
4. Redactá correos

ANTES de mandar nada, mostrame el lote en un mensaje para aprobar.

Si apruebo → mandá vía mi cuenta conectada.
Si NO apruebo → descartá.

Avisame cuántos créditos de Apify llevo gastados en el mes para
no pasarme del plan gratis.
```

## Anti-patrones

### 1. Saltearse la entrevista (Prompt 1)

Sin entrevista → leads genéricos → conversión baja → "Apify no funciona". El problema es el **brief**, no la herramienta.

### 2. Mandar 50 correos idénticos

Aunque uses Apify, si tu correo es copy-paste, **te van a marcar como spam**. Personalización ES el punto.

### 3. Quemar créditos en sites equivocados

Sacar 1000 leads de LinkedIn cuando vendés a negocios locales = créditos perdidos. **Elegí la red correcta primero**.

### 4. Sin auth en sitios privados

Algunos actors requieren login. **Guardá credenciales en .env** o en la integración oficial — nunca en el prompt.

### 5. Scrapear sin respetar ToS

Apify maneja anti-bot bien, pero **algunos sites prohíben scraping comercial**. Verificá ToS antes de operar a gran escala.

## Compliance y ética

### Lo que SÍ podés

- ✅ Scrapear datos **públicos**
- ✅ Contactar empresas para B2B legítimo
- ✅ Construir tus propios datasets para análisis

### Lo que NO

- ❌ Scrapear data **privada / login required** sin autorización
- ❌ Spam masivo no solicitado a particulares
- ❌ Violar GDPR (EU) o CCPA (California)
- ❌ Vender datos personales sin consentimiento

**Cuando hay duda → consultá**. Algunas industrias (legal, médica, finanzas) tienen reglas estrictas.
