```
  - copywriting
```

### In Claude Code

```bash
# Install all
npx skills add coreyhaines31/marketingskills

# Or selectively
npx skills add coreyhaines31/marketingskills --skill copywriting
npx skills add coreyhaines31/marketingskills --skill page-cro
```

## How to use them

**You don't need to invoke the skill by name**. Claude detects it based on your prompt.

### Example 1 — Landing page copy

```
> I need copy for a B2B SaaS inventory landing page.
  Audience: COOs at mid-size retail in LATAM.

  Generate:
  - Hero (headline + subhead)
  - 3 feature sections
  - Main CTA
```

Claude uses `copywriting` + `page-cro` + `product-marketing` automatically.

### Example 2 — Growth strategy

```
> My SaaS hit $15k MRR. I want to get to $50k in 6 months.
  Current stack: organic content + paid Google.

  Strategy to grow 3x while keeping current CAC.
```

Claude uses `growth-strategy` + `pricing-strategy` + `meta-ads`.

### Example 3 — Email sequence

```
> A 5-email welcome sequence for new signups to my SaaS.
  Goal: convert free trial to paid within 14 days.
```

Claude uses `email-marketing` + `welcome-sequence` + `retention`.

## Anti-patterns

### 1. Installing all 33 at the start

Your Claude gets distracted with 33 active skills. **Start with 5-7** relevant to your role.

### 2. Expecting every output to be great

Skills are a **knowledge base**. Your **specific brief + iteration** is what makes them shine.

### 3. Forgetting context

Each skill applies differently depending on your industry, audience, and stage. **Give it** specific context, always.

### 4. Not iterating

The first generation is 70%. Iterating gets you to 95%.

### 5. Confusing tactical with strategic

Skills cover the tactical (how to do email X). For real strategic decisions, **complement with humans**.
