## La idea

Claude es persona muy inteligente **encerrada en un cuarto**. Estas 4 herramientas le abren **4 ventanas al mundo**:

| Superpoder | Para qué |
|------------|----------|
| **Supadata** | Ve y entiende videos de cualquier red social |
| **Apify** | Busca y extrae datos de toda internet |
| **Last 30 Days** | Lee últimas noticias de cualquier nicho |
| **Playwright CLI** | Navegador propio para investigar web |

- **Claude Code** instalado

## Superpoder 1 — Supadata (ver videos)

### Qué hace

API que **transcribe videos completos** + da métricas (vistas, likes, comentarios). Soporta YouTube, TikTok, Instagram, prácticamente cualquier red.

Le pasás link → Claude obtiene todo el contenido del video en texto, listo para analizar.

### Setup

1. Cuenta gratis en `supadata.ai`
2. Copiá API key del dashboard
3. Configurá en `.mcp.json`:

```json
{
  "mcpServers": {
    "supadata": {
      "command": "npx",
      "args": ["-y", "@supadata/mcp"],
      "env": {
        "SUPADATA_API_KEY": "tu-key"
      }
    }
  }
}
```

### Casos de uso

- **Analizar videos de competencia**: qué dicen, cómo, qué funciona
- **Estructura de videos virales**: descomponer hooks + body + payoff
- **Extraer ideas de tutoriales**
- **Métricas sin abrir la plataforma**

```
> Analizá este video viral: [URL TikTok]

  Decime:
  1. Estructura (hook + body + payoff)
  2. Por qué funciona psicológicamente
  3. Cómo podría replicar en mi nicho [TEMA]
```

## Superpoder 2 — Apify (scrapear internet)

### Qué hace

1500+ scrapers ya armados (actors) para Google Maps, LinkedIn, Instagram, Facebook, Amazon, etc. Conectado a Claude por MCP.

Ver [Apify clientes ilimitados](/boveda/apify-clientes-ilimitados) para profundizar.

### Setup rápido

URL del MCP: `mcp.apify.com`

Settings → Connectors → Add MCP → pegás URL + tu API token.

### Casos de uso

- **Listas de leads** (Google Maps, LinkedIn)
- **Monitor de competencia** (precios, posts, ads)
- **Research masivo** estructurado

```
> Sacame 50 [profesión] de [ciudad] con teléfono y email
  usando Apify Google Maps Scraper.
```

### Casos de uso

```

```


## Superpoder 4 — Playwright CLI (navegador propio)

### Qué hace

Le da a Claude **navegador real** (Chrome). Puede:
- Visitar URLs
- Hacer clicks
- Llenar forms
- Tomar screenshots
- Scrolear, hover, etc.

### Instalación

```bash
claude mcp add playwright -- npx -y @anthropic-ai/playwright-mcp@latest
```

100% gratis sin API key.

### Casos de uso

```
> Abrí [URL] con Playwright. Hacé login con [credenciales].
  Navegá a /dashboard. Exportá CSV de "Last 30 days". Guardalo
  en /data/.
```

```
> Probá flow de signup de mi propio site http://localhost:3000.
  Hacé screenshots de cada paso. Reportá si algo se rompe.
```

Ver [navega y automatiza](/boveda/navega-y-automatiza) para Firecrawl + Playwright combinados.

## Anti-patrones

### 1. Instalar 4 y no usar 3

Cada herramienta tiene costo de mantenimiento mental. **Instalá según necesidad real**.

### 2. Abusar de Supadata/Apify gratis

Tier free tiene límites. Si lo usás mucho, **upgradeá** o vas a tener throttling.

### 3. Esperar precisión perfecta

Supadata transcribe **muy bien** pero no perfecto. Apify scrape **muy bien** pero algunos sites bloquean. Aceptá la realidad.

### 4. No respetar ToS

Scrapear sites comerciales puede violar terms. **Validá antes** de operar a escala.

### 5. Sin disclosure cuando uses data scraped

Si publicás insights basados en data de competencia, **anonimizá** o pedí permiso si vas a citarlos.
