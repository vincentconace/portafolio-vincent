## Tool 1 — Awesome Claude Skills

### Analogy

Claude is like a chef who knows how to cook, but **without recipes**. Awesome Claude Skills is the recipe book. You hand it the book and suddenly it knows how to make scroll-stopping hooks.

### What it does

- Ready-made skills for writing viral content
- Scroll-stopping hooks
- Proven video structures
- The "hook + body + payoff + CTA" pattern

Repo: `ComposioHQ/awesome-claude-skills`. Open source.

### Install

```
> Go into this repository:
  https://github.com/ComposioHQ/awesome-claude-skills

  Find the viral content and scriptwriting skills.
  Install the one that helps with writing hooks and structuring
  viral videos.

  Use that skill for everything I ask you from now on.

## Tool 2 — UltraThink

### Analogy

When you ask it something, Claude **responds fast** but doesn't stop to think it through much. Like asking a friend a question while they're walking — you get a quick answer.

UltraThink is like telling it: **"Sit down and think it through before you answer"**.

### What it does

- Deep reasoning
- Compares different angles
- The difference between a boring script and a well-structured one

### How to use it

A single word at the start of the prompt:

```
ultrathink — write me a viral script for [TOPIC] in a 45s format
for TikTok.
```

**You don't need to install anything**.

## Tool 3 — VoxCPM (voice cloning)

### What it does

- Clone your voice **in seconds**
- 30 languages supported
- Studio quality
- Free, open source

### Setup

```
> Install VoxCPM on my computer.

  To clone my voice:
  1. Take a sample audio (30s of my voice speaking clearly)
  2. Process it
  3. Generate my vocal clone
  4. Test it with a test phrase
```

### Use

```
> Take the script we generated and use VoxCPM with my cloned voice
  to generate the Reel audio.

  Tone: energetic at the start, calm at the payoff.
  Output: /audio/reel-[DATE].mp3
```

## The full flow

### Brief (30 seconds)

```
> ultrathink — I need a 45s Reel about [TOPIC].

  Audience: [TARGET]
  Goal: [educate / entertain / sell]
  Platform: [TikTok / Reels / Shorts]

  Use Awesome Claude Skills for the viral structure.

  Generate:
  1. Hook (first 3 sec, must stop the scroll)

  3. Payoff (reveal / conclusion)

### Output (~2-3 minutes)

- MP3 audio with your voice

### Assemble into video

You drop the audio into CapCut or [Editor Pro Max](/boveda/editor-pro-max) with visuals. Total: **10-15 min for a publishable Reel**.

## Anti-patterns

### 1. Using UltraThink for everything

`ultrathink + give me a date` = overhead. Only for reasoning that genuinely justifies it.

### 2. Expecting perfect audio on the first try

VoxCPM is good but not perfect. **Iterate**. If the tone doesn't convince you, regenerate with a more specific instruction.

### 3. Not humanizing the script

Even with Awesome Claude Skills, the output can sound like AI. **Run it through [humanízalo](/boveda/humanizalo)** before generating audio.

### 4. Cloning your voice with poor-quality audio

If your 30s sample has background noise, **the clone will sound off**. Record clean.

### 5. Publishing without reviewing

The workflow is fast but **always review** before publishing. Pronunciation errors, odd words, etc.

---

## Ethical limitations

### Voice cloning

VoxCPM clones YOUR voice. To clone **someone else's** voice you need explicit permission. Otherwise it's a **deepfake**.

### Disclosure

On some platforms (TikTok, YouTube), if you use an AI voice, **disclose it**. The rules change — check.
