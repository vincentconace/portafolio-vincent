**No coding required**. The whole process is clicks and permissions.

## App vs. browser difference

- **claude.ai (web)** = chat only. No tool connections.
- **Claude Desktop** = app. **It DOES connect to Gmail, Calendar, Drive, files**.

For this use case, **you need the app**.

---

## Setup in 3 steps

### 1. Download Claude Desktop

`claude.ai/download` → download for your OS → install → log in with your Anthropic account.

### 2. Connect Gmail

1. Settings (gear icon)
2. **Connections** or **Integrations**
3. **Google Workspace** (includes Gmail + Calendar + Drive)
4. The Google window asks for permissions → **Allow**

### 3. Verify

```
> List my 5 most recent unread emails.
```

If it returns the list, you're connected.

## 6 copy-paste prompts

### 1. Organize by topic

```
> Review my last 50 emails and group them by topic: work,
  personal, invoices, newsletters, other.
  Give me a summary of each group.
```

### 2. Flag urgent emails

```
> From my unread emails, identify which ones need an urgent
  reply (today) and which can wait.
  Explain why.
```

### 3. Summarize long conversations

```
> Summarize the 3 longest email threads from this week.
  Give me key points + whether there's an action pending on my end.
```

### 4. Find info (invoices, receipts)

```
> Search my emails from the last month for any invoice or receipt.
  Give me a list with date, sender, and amount.
```

### 5. Reply drafts

```
> Review my unanswered emails from the last 3 days.
  Draft short, professional replies for each one.

  DO NOT send them. Just drafts for me to approve.
```

### 6. Subscription cleanup

```
> Identify newsletters, promos, and automated emails I receive
  repeatedly.
  Give me a list so I can decide which to unsubscribe from.
```

## Security

| Guarantee | Details |
|----------|---------|
| Permission per action | Claude asks for confirmation before touching anything |
| No sending without your OK | It drafts replies, you decide to send |
| Privacy | Anthropic does NOT use your emails for training |
| Disconnect | Settings → Connections → Revoke at any time |

### Be careful with sensitive info

Even though Claude is secure, **use common sense**:
- Avoid processing emails with passwords, full banking details, medical info
- If an email has very sensitive data, **handle it yourself** directly

## Tips for better results

### 1. Be specific

❌ "Organize my email"
✅ "Group this week's emails by sender and tell me which ones need a reply"

### 2. Start small

Don't try to process 10,000 emails at once. **Last 7 days first**, then expand.

### 3. Review before sending

Claude shows the drafts before sending. **Always review them**. Sometimes it over-formalizes or misses nuance.

### 4. Weekly maintenance

Monday, 5 min: ask Claude for a summary of the week + pending actions. Your inbox stays clean effortlessly.

### 5. Multiple accounts

You can connect **several Gmail accounts**. When you ask for something, Claude asks which one to use.

  - Number of emails received vs. replied to

  - Threads with commitments I haven't closed yet
  - Suggestion of what to prioritize in June

## Anti-patterns

### 1. Auto-sending replies

NO. Even if Claude writes well, **always review**. A misinterpreted reply can damage relationships.

### 2. Trusting the classification 100%

Claude may mark something critical as "not urgent" if the context didn't show it. **Review the flags**.

### 3. Processing emails with sensitive legal/financial data

For emails with contracts, banking details, medical info → handle them yourself. Not Claude.

### 4. Not iterating on instructions

If Claude classifies something wrong, **teach it**: "this type of email goes to 'support', not 'urgent'". It improves with feedback.

### 5. Applying the same rules to personal + work

Connect separate accounts with different rules. **Don't mix contexts**.
