## What it is

"Awesome Claude Code Plugins" is the "awesome list" for Claude Code: a curated collection of slash commands, subagents, MCP servers and hooks. The key difference from other large marketplaces is that here the curation is more careful — it's not "everything out there," it's "what's actually worth it."

The most important part: it includes Anthropic's official plugins in the catalog. If you want to try what Anthropic publishes as a reference without having to hunt for it in separate repos, this is the shortcut.

It covers 118 plugins in total, with a focus on PR workflows, code review, debugging, git, planning, security, and a solid set of agents.

## What you can do with it

### Start with the official ones

If you haven't installed anything from the community yet and want to play it safe, the three Anthropic plugins (`agent-sdk-dev`, `pr-review-toolkit`, `commit-commands`) are the most conservative base set. You can layer others on top afterward if you want.

### PR review from six angles

`pr-review-toolkit` runs six different agents over your PR: one reviews comments, another tests, another error handling, another type design, another code quality, another simplification. It's more thorough than asking Claude to "review this PR" in a single pass.

### Passive security hook

`security-guidance` stays in the background and warns you if an edit introduces something risky (command injection, XSS, insecure patterns). It doesn't block — it alerts. A good guardrail so you don't ship obvious mistakes to prod.

### Combine with Galen Ward's planning

`plan` and `explore` set up the Explore → Plan → Execute pattern. Combine them with `pr-review-toolkit` at the end and you get a guided feature → review → merge workflow.

<Callout type="warning" title="Before installing">

- **If you've already run [Claude Code Setup](/boveda/claude-code-setup),** you already received Anthropic-style recommendations. This marketplace expands on that but partially overlaps.
- **Overlap with `cc-marketplace`:** both share many plugins (Anand's, Galen's, Galpert's). If you're going to use one, don't install both.
- **Don't expect a single thematic curation.** It's a mix of areas — an advantage if you want variety, a disadvantage if you're looking for a single focus.
