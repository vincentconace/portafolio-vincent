## Qué es Scrapling

Imaginá que internet es un supermercado gigante y querés lista de todos los yogures con nombre, precio y marca. **Scrapling es tu robot** que va pasillo por pasillo, anota cada yogur, te arma la lista en Excel. **Las tiendas no se dan cuenta** que fue un robot.

Técnicamente: librería open source de Python que extrae info de cualquier web (Google Maps, directorios, redes sociales, tiendas) **sin que te bloqueen**. Se camufla como navegador humano real.

| Detalle | Valor |
|---------|-------|
| GitHub | 37,500+ ★ |
| Licencia | BSD-3 (gratis hasta comercial) |
| MCP integrado | ✅ Claude lo controla directo |
| Camuflaje | Camoufox (anti-detección) |
| Costo real | **$0** |

Repo: `D4Vinci/Scrapling`.

## Instalación — un prompt y Claude hace todo

**No vas a instalar a mano**. Le pegás el prompt a Claude Code o Desktop y él se encarga.

### Prompt maestro

```
Quiero usar Scrapling para sacar listas de clientes. Es
librería Python de D4Vinci con MCP integrado para Claude.
Repo: https://github.com/D4Vinci/Scrapling

Hacé todo esto reportando cada etapa:

1. Verificá que tengo Python 3.10+ y uv instalados.
   Si falta alguno, decime cómo instalarlo.

2. Instalá Scrapling con dependencias completas:
   uv pip install "scrapling[all]"

3. Corré instalación de browsers que Scrapling necesita
   (Camoufox, Playwright, etc):
   scrapling install

4. Leé el README oficial para encontrar el comando exacto del
   server MCP (suele ser "scrapling-mcp" o "scrapling mcp serve").

5. Agregá Scrapling al claude_desktop_config.json dentro de
   "mcpServers". Si tengo otros MCPs, NO los sobrescribas.
   Ubicaciones:
   - macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
   - Windows: %APPDATA%\Claude\claude_desktop_config.json

6. Probá la instalación: sacá los primeros 3 resultados de
   Google para "dentistas cdmx" y mostramelos.

7. Decime qué hago: reiniciar Claude Desktop + abrir conversación
   nueva.

Si algo falla, explicame qué salió mal y cómo arreglarlo —
NO sigas al siguiente paso sin resolver.
```

### Importante post-instalación

Después del paso 5, **cerrá Claude Desktop COMPLETO** y reabrí:
- **macOS**: Cmd+Q
- **Windows**: bandeja del sistema → Salir

Cerrar la ventana no basta. Claude necesita leer la nueva config para detectar el MCP.

## 3 casos de uso reales

### Caso 1 — Lista de nicho local

Agencias cobran $500 por lista así. Vos la sacás en 5 min.

```
Usá Scrapling para armar lista de 50 [profesión — ej:
veterinarias, contadores, abogados, mecánicos] en [ciudad].

Para cada uno: nombre, teléfono, dirección, rating, número
reseñas, sitio web (si tienen) y link directo a la ficha
Google Maps.

Limpiá duplicados. Filtrá los con menos de 3 reseñas (suelen
ser falsos).

CSV ordenado de mayor a menor por reseñas (los más activos
primero), guardado en Desktop como
"leads-[profesión]-[ciudad].csv".
```

### Caso 2 — Negocios SIN presencia web

Si vendés diseño web, dev o ads, **este es oro puro**.

```
Usá Scrapling para sacarme 100 restaurantes en [ciudad] de
Google Maps.

Para cada uno verificá si tienen sitio web propio (NO cuenta
link a Facebook, Instagram ni TripAdvisor — solo dominio propio).

Quedate SOLO con los que NO tienen sitio web.

Para esos quiero: nombre, teléfono, dirección, horario, rating,
número reseñas, link Google Maps.

Ordenalos de mayor a menor por reseñas — los más populares sin
web son los que más rápido cierran.

CSV en Desktop como "restaurantes-sin-web-[ciudad].csv".
Decime cuántos encontraste del total.
```

### Caso 3 — Monitor de precios competencia

Servicios de "price intelligence" cobran $500-2000/mes. Acá lo hacés gratis.

```
Usá Scrapling para extraer precios actuales de estos productos
de mi competencia:

[Pegá URLs de competencia, una por línea]

Para cada producto: nombre, precio actual, descuento si hay,
stock, URL, fecha de extracción.

Compará con mis precios: [opcional: ruta a tu catálogo]

Decime:
- Qué productos tengo más caros (pierdo ventas)
- Qué productos tengo demasiado baratos (dejo dinero)
Prioridad por mayor diferencia.

Guardá como "competencia-[YYYY-MM-DD].csv" en Desktop.

Si quiero que corra cada semana automático, decime cómo
agendarlo.
```

```
Tomá este CSV: [ruta] y:

2. Normalizá teléfonos a formato +52 XX XXXX XXXX (o de [país])
```

```
directorios locales. Si tampoco, marcá "no encontrado" en

```

```
Con Scrapling, buscá cuentas de Instagram de [tipo negocio —

Para cada cuenta: @usuario, nombre mostrado, bio, seguidores,
```

## Trucos avanzados

### Stealth mode

Scrapling usa **Camoufox** (browser anti-detección). Pedile a Claude que lo active para sites con Cloudflare strict.

### Rotación de proxies

Para listas grandes (1000+ registros), configurá proxies rotativos. Claude puede integrarlos con MCP de proxy pool.

### Auto-match selectors

Scrapling detecta cambios en HTML de un site y **actualiza selectores solo**. Tus scrapers no rompen aunque el site cambie.

## Scraping ético — tu negocio depende

### Respetá robots.txt + ToS

Si un site dice "no scraping", **no scrapees**. Algunas violaciones pueden traer demandas.

### Rate limiting

Scrapling ya espera entre requests. **NO lo aceleres**. Quemar un site te quema a vos.

### No toques data personal sensible

❌ Direcciones privadas, datos médicos, info privada
✅ Contactos comerciales de Google Maps (públicos, el negocio mismo los publicó para que lo contactes)

### GDPR / LFPDPPP

Si contactás los leads:
- Identificate
- Explicá de dónde sacaste sus datos
- Ofrecé opt-out

**Email transparente vende 10x más que spam**.

---

## Anti-patrones

### 1. Acelerar el rate limiting

Querer 10x velocidad → te banean → pierdes acceso. **Paciencia**.

### 2. Scrapear LinkedIn pesado con Scrapling

LinkedIn detecta y bloquea fácil. Para LinkedIn → [Apify](/boveda/apify-clientes-ilimitados).

### 3. Vender lista sin contactarlos

Lista sin contacto efectivo = pérdida de tiempo. **Vendé el outreach también**, no solo la lista.

### 4. Sin disclaimer al contactar

Si contactás sin decir de dónde sacaste el dato, **spam puro**. Disclaimer + transparencia.

### 5. Saltarse el setup del MCP

El MCP es lo que hace mágico el flow. Sin MCP, podés usar Scrapling como librería normal pero perdés la integración con Claude.
