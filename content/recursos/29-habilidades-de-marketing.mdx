## Qué es

Imaginate contratar a **33 especialistas de marketing** — uno para SEO, otro para copywriting, otro para ads, otro para email — y tenerlos 24/7.

Eso son estos skills: **playbooks de experto** que le dan a Claude conocimiento especializado en cada área del marketing.

- **Instalás una vez** → quedan para siempre
- **Lenguaje natural**: decís qué necesitás, Claude detecta el skill correcto
- **Open source MIT** (Corey Haines)

Repo: `coreyhaines31/marketingskills`.

## Instalación

### En Claude (web)

```
> Instalá la colección awesome-claude-skills:
  https://github.com/coreyhaines31/marketingskills

  Para cada skill, hacé el upload correspondiente desde
  Settings → Skills.

  Empezá con los 5 más usados:
  - copywriting
  - email-marketing
  - page-cro
  - meta-ads
  - growth-strategy
```

### En Claude Code

```bash
# Instalar todas
npx skills add coreyhaines31/marketingskills

# O selectivas
npx skills add coreyhaines31/marketingskills --skill copywriting
npx skills add coreyhaines31/marketingskills --skill page-cro
```

## Cómo usarlas

**No necesitás invocar el skill por nombre**. Claude detecta basándose en tu prompt.

### Ejemplo 1 — Copy de landing

```
> Necesito copy para landing de SaaS B2B de inventario.
  Audiencia: COOs de retail mediano LATAM.

  Generá:
  - Hero (headline + subhead)
  - 3 secciones de features
  - CTA principal
```

Claude usa `copywriting` + `page-cro` + `product-marketing` automático.

### Ejemplo 2 — Estrategia de growth

```
> Mi SaaS llegó a $15k MRR. Quiero pasar a $50k en 6 meses.
  Stack actual: organic content + paid Google.

  Estrategia para crecer 3x manteniendo CAC actual.
```

Claude usa `growth-strategy` + `pricing-strategy` + `meta-ads`.

### Ejemplo 3 — Email sequence

```
> Welcome sequence de 5 emails para nuevos signups de mi SaaS.
  Goal: convertir trial gratuito a paid en 14 días.
```

Claude usa `email-marketing` + `welcome-sequence` + `retention`.

## Anti-patrones

### 1. Instalar las 33 al inicio

Tu Claude se distrae con 33 skills activas. **Empezá con 5-7** relevantes a tu rol.

### 2. Esperar que cada output sea genial

Skills son **base de conocimiento**. Tu **brief específico + iteración** es lo que las hace brillar.

### 3. Olvidar contexto

Cada skill aplica diferente según tu industria, audiencia, etapa. **Dale contexto** específico siempre.

### 4. No iterar

Primera generación es 70%. Iterar te lleva a 95%.

### 5. Confundir tactical con strategic

Skills cubren tactical (cómo hacer email X). Para decisiones estratégicas reales, **complementá con humanos**.
