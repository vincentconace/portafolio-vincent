## The problem

When you tell an image generator "a cat in a garden", **it has to guess everything else**:
- What style? (photo, illustration, anime, 3D?)
- What lighting? (midday, golden hour, dramatic?)
- What angle? (eye-level, high angle, low angle?)
- What textures?
- What mood?

Result: **generic image**, flat lighting, no personality.

**Claude Banana** is a project that takes your simple prompt and **rewrites it with 7 ingredients** that generators need to deliver professional images.

Repo: `Hainrixz/claude-banana`. Open source.

---

## The formula — 7 ingredients

| # | Ingredient | What it's for |
|---|-------------|----------|
| 1 | **Subject** | What appears (person, object, scene) |
| 2 | **Visual style** | Photo / illustration / 3D / cinema / watercolor |
| 3 | **Environment** | Studio / nature / city / abstract |
| 4 | **Lighting** | Golden hour / dramatic / neon / natural |
| 5 | **Action** | Movement, pose, interaction |
| 6 | **Camera angle** | Close-up / top-down / eye-level / wide angle |
| 7 | **Textures** | Skin, fabric, metal, wood, water |

**Important**: current generators (Gemini, Flow, etc.) prefer **descriptive sentences**, not comma-separated keyword lists. Claude Banana generates in **narrative format**.

## Setup — under 1 minute

### Step 1 — Download

`github.com/Hainrixz/claude-banana` → Download ZIP.

Unzip it.

### Step 2 — Open with Claude

**Option A — Desktop app**:
1. Open Claude Desktop
2. Projects → New Project
3. Select the unzipped folder

**Option B — Claude Code in terminal**:
```bash
cd claude-banana-main
claude
```

### Step 3 — Write your request

```
> I want an image for my new SaaS launch.
  Concept: person looking at a modern dashboard, productive
  but calm mood.
```

### Step 4 — Paste the output into a generator

Claude Banana hands you an optimized prompt. You copy it and paste it into:
- **Flow** (Google Labs) — free
- **Whisk** (Google Labs) — free for remixing
- **Midjourney**
- **DALL-E**
- **Stable Diffusion**

Works with any modern generator.

## The 9 specialized modes

Each mode tweaks the formula for a specific type of image:

| Mode | What it's for |
|------|----------|
| **Cinema** | Cinematic frames with dramatic lighting |
| **Product** | Product photos for e-commerce and catalogs |
| **Portrait** | Professional portraits with studio lighting |
| **Fashion** | Fashion editorial with styling and composition |
| **UI Design** | Assets for interfaces and digital design |
| **Logos** | Logos, icons, visual branding |
| **Landscapes** | Nature, cities, panoramic scenes |
| **Abstract** | Generative art, patterns, free compositions |
| **Infographics** | Data visualizations, stylized diagrams |

To invoke a mode:

```
> Use Cinema mode. Image: detective in a 1940s bar
  smoking, dramatic shadows.
```

```
> Product mode.

  Product: black running shoes with lime green accents.

  I need 3 variants:
  - 3/4 view with a soft gray background (catalog)
  - Side view with macro textures (material detail)
  - In-use view (running, golden hour, urban)

> Cinema mode.

  Landing page for a B2B retail inventory SaaS.
  Audience: mid-size retail COOs in LATAM.

  I need a hero with a professional person (40-50 years old),
  confident but approachable expression, in a modern office,

  Mood: control + technology + human (not cold corporate).
  Palette: deep blues + warm accents.

```
> Logos mode.

  Business: a specialty coffee shop called "Compass".

  Style: minimalist, geometric, sophisticated but approachable.
  Colors: espresso coffee brown + cream + gold accent.
  Inspired by: brands like Aesop, Blue Bottle Coffee.

## Key rules it applies

### Narrative format

Generators understand **complete sentences better** than comma-separated keyword lists.

❌ "cat, garden, golden hour, watercolor, eye-level, lavender"
✅ "A tabby cat lounging in a cottage garden at golden hour..."

### No forbidden words

It does NOT use terms that add nothing:
- "4K"
- "masterpiece"
- "hyperrealistic"
- "ultra detailed"

These terms **don't improve** modern results and can make them worse.

### Compatible with everything

Works with Gemini, Midjourney, DALL-E, Stable Diffusion. Narrative works across the major engines.

### Easy iteration

If you don't like the output, you don't start from scratch:

```
> The cat from the previous prompt, but darker and with neon
  cyan/magenta light instead of golden hour.
```

Claude adjusts only what you asked for and keeps the rest.

## Anti-patterns

### 1. Hyper-long prompts

Past 200 words, generators **get confused**. If Claude Banana generates a long paragraph, that's fine — but beyond that, refine.

### 2. Mixing too many styles

❌ "Realistic photo + watercolor + 3D + cinematic + anime"

Pick **one dominant style**, not 5.

### 3. Using Claude Banana for UI designs

For functional UIs (buttons, dashboards), use [Stitch](/boveda/stitch-diseno-ia) or the [web designer](/boveda/claude-disenador-web). Claude Banana is for **images/art**, not functional interfaces.

### 4. Not iterating

The first generation is rarely perfect. **Iterate** with specific feedback. It's the difference between amateur (accepts the first) vs pro (refines 3-5 times).

### 5. Expecting pixel-perfect

Generators have **variance**. The same prompt can produce slightly different images. Accept the variance or use seeds (if the generator supports them).
