## El problema

Cada semana sale herramienta nueva. Probás 3, te quedás con 2, abandonás 1, volvés a la siguiente. Resultado: **5 pestañas abiertas, 3 suscripciones que no usás, viernes a la noche** intentando acordarte cuál era la buena.

**Un Reel de 1 min te toma 4 horas** porque saltás entre Claude, Gemini, Midjourney, Runway, Descript, CapCut, Canva, Notion. Cada salto = copy-paste, re-prompt, cuenta nueva. Para cuando lo subís, ya pasó la idea.

**La solución**: stack chico, fijo, donde cada herramienta hace **una cosa específica** y todas se conectan con **un solo cerebro arriba** — Claude para texto/dirección + Master Prompt que genera prompts perfectos para cada modelo.

| Antes | Después |
|-------|---------|
| 15 tabs · 6 cuentas · 0 stack | 6 tools · 1 Master Prompt |
| Re-elegís tool a cada paso | Cada tool tiene rol fijo |
| **4 horas por Reel** | **40 min por Reel** |

## El mapa — 6 eslabones de una cadena

### 1. Guion → Claude (Opus 4.7 + Sonnet)

Convierte idea cruda en **guion humanizado de 45-55s** con hook + body listicle + payoff + CTA.

- **Opus 4.7** para razonamiento estructural (cómo encadenar argumentos)
- **Sonnet** para iteración rápida de variantes

### 2. Imagen → Nano Banana Pro + ChatGPT Image

- **Nano Banana Pro**: velocidad para drafts y B-roll estático
- **ChatGPT Image**: detalle photoreal para portadas

### 3. Video → Veo 3.1 + SeeDance 2

- **Veo 3.1**: realismo cinematográfico
- **SeeDance 2**: velocidad para iteración

Lo que era un día de rodaje, **10 minutos**.

### 4. Voz → ElevenLabs

- Clonación de voz en **30 segundos**
- Voces de personajes para narración, doblaje, ASMR de marca

### 5. Avatar → HeyGen + Higgsfield

- **HeyGen**: tu avatar UGC hablando a cámara
- **Higgsfield**: efectos cinematográficos image-to-video

Ver [Higgsfield MCP](/boveda/higgsfield-mcp) para integración directa.

### 6. DMs → ManyChat

Automatiza respuestas a comentarios y DMs. Si tu Reel viraliza, ManyChat **convierte audiencia en leads**.

## El Master Prompt

Arriba de todas las herramientas. Es lo que las une.

Vos solo le decís a Claude:

```
> Idea: [concepto]
> Objetivo: [educar / entretener / vender]
> Audiencia: [target]
> Plataforma: [TikTok / Reels / YouTube]

Generame:
1. Guion 45-55s con hook + 3 puntos + payoff + CTA
2. Prompt para Nano Banana (portada)
3. Prompt para Veo 3.1 (B-roll principal)
4. Prompt para ElevenLabs (tono de voz)
5. Prompt para HeyGen (postura + gesto)
6. Mensaje de DM automática para ManyChat
```

Claude genera **los 6 prompts listos** para cada herramienta.

## Workflow real (40 min)

### 0-5 min — Brief

Le tirás a Claude la idea + audiencia + formato.

### 5-15 min — Master Prompt

Claude genera:
- Guion humanizado (pasa por humanizer skill)
- Prompts para cada herramienta del stack

Vos validás. Ajustás 1-2 cosas.

### 15-25 min — Activos visuales

Mientras Claude piensa, en paralelo:
- Nano Banana → portada
- Veo 3.1 → 3 clips B-roll
- HeyGen → tu avatar hablando

### 25-30 min — Voz + audio

ElevenLabs lee el guion con tu voz clonada.

### 30-35 min — Ensamble

Editor (CapCut, Descript) o [Editor Pro Max](/boveda/editor-pro-max). Pegás todo.

### 35-40 min — Upload + ManyChat

Subís Reel/TikTok. Activás flujo de ManyChat para responder DMs.

## Master Prompt — versión completa para copiar

```
Sos director creativo IA para creator [TU NICHO].

Idea: [CONCEPTO CRUDO]
Objetivo: [EDUCAR / ENTRETENER / VENDER]
Audiencia: [TARGET ESPECÍFICO]
Plataforma: [TIKTOK / REELS / YOUTUBE SHORTS]
Duración: 45-55s

Generame:

1. GUION HUMANIZADO
   - Hook (3 segundos primeros)
   - 3 puntos del cuerpo (listicle style)
   - Payoff (revelación / conclusión)
   - CTA clara
   - Pasalo por humanizer mental (NO suene IA)

2. PROMPT PARA NANO BANANA PRO (portada)
   - Estilo visual específico
   - Composición
   - Paleta
   - Mood

3. PROMPT PARA VEO 3.1 (B-roll principal 5-10s)
   - Escena cinematográfica
   - Cámara (movimiento, ángulo)
   - Iluminación
   - Sujeto + acción

4. PROMPT PARA ELEVENLABS (tono de voz)
   - Emoción dominante
   - Ritmo (rápido / pausado)
   - Énfasis en frases clave del guion

5. PROMPT PARA HEYGEN (avatar UGC)
   - Postura
   - Gesto / expresión
   - Vestuario
   - Background

6. MENSAJE PARA MANYCHAT (auto DM)
   - Cuando alguien comente [TRIGGER]
   - Mandar [MENSAJE ESPECÍFICO]
   - Con [CTA / link]

Entregá los 6 listos para copy-paste a cada herramienta.
```

## Anti-patrones

### 1. Querer probar TODO antes de empezar

Si esperás "probar todas" terminás sin producir. **Empezá con Claude + ElevenLabs + ManyChat** (~$57/mes). Sumás cuando justificás.

### 2. Saltar el Master Prompt

Si usás cada herramienta sin orquestación, **volvés al zoológico**. El Master Prompt es lo que vuelve esto **stack**, no colección.

### 3. No humanizar el guion

El guion generado por Claude **necesita humanizer**. Si publicás output crudo, **suena a IA**. Pasalo por [humanizalo](/boveda/humanizalo).

### 4. Avatar HeyGen sin entrenarlo bien

Avatar mediocre = arruinás el video. **Entrenalo con buena luz, varios ángulos, audio limpio**.

### 5. ManyChat sin flujo claro

Si la DM no tiene CTA específico, **es ruido**. Cada DM debe llevar a algo (link, lead magnet, calendly).
