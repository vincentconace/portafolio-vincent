## What it is

JFrog is one of the most widely used enterprise platforms for artifact management and software supply chain: Artifactory for artifacts (Docker, npm, Maven, etc.), Xray for security scanning, Distribution for release management.

This MCP gives Claude access to those services. The inline description on aitmpl is generic — the exact detail of the exposed tools is in the [configuration JSON](https://github.com/davila7/claude-code-templates/blob/main/cli-tool/components/mcps/devtools/jfrog.json).

Useful for enterprise teams where JFrog is the source of truth for artifacts and supply chain.

## How to install it

```bash
npx claude-code-templates@latest --mcp devtools/jfrog
```

Prerequisite: a JFrog account and an access token.

For Claude:

```
I installed the devtools/jfrog MCP. Show me which tools the server exposes and
list the Artifactory repositories my user can see.
```

## What you can do with it

### Artifact inspection

"List the artifacts in the `docker-prod` repo sorted by last update. For the 5 most recent ones, give me metadata."

### Security findings with Xray

"Find critical vulnerabilities in the artifacts published in the last week, grouped by component." Useful for security triage.

### Version lookup

"In the `libs-release` repo, which versions of `my-lib` have been published and since when?" History to understand the lifecycle.

### Usage audit

"Which artifacts haven't been downloaded in the last 90 days? Candidates for archiving."

### Integration with release management

If your pipeline publishes to JFrog Distribution, the MCP lets you query release status from Claude.

## Prerequisites

- **JFrog account** with a suitable plan
- **Access token** with permissions over the relevant services
- **Check the configuration JSON** to understand exactly what it exposes

## When it's NOT a good fit

<Callout type="warning" title="Before installing">

- **If you don't use JFrog:** obviously. For Docker Hub, GitHub Packages, etc., other MCPs or direct tooling.
- **Broad permissions:** an admin token lets Claude operate everything. Restrict the scope.
- **For code quality analysis:** [Codacy](/boveda/mcp-codacy) is more focused on that. JFrog Xray is for supply chain security.
- **Costs:** JFrog Enterprise is paid. Validate your plan before heavy use.
- **Only 78 installs:** a small community testing it. Try it in a sandbox first.

</Callout>
